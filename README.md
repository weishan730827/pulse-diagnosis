## 张锡纯辨证方法论——原著提炼的完整决策树

> 源于《医学衷中参西录》原著，非网络解读。张锡纯辨证的根本方法是：以"阴常不足"为病机观，以气机升降为框架，以三部总看脉法定虚实，以左右对比定脏腑，最终在自创方和仲景方两大方池中精准锁定。

### 根本病机观

**已病十之八九阴常不足，阳常有余。** 阳常有余非真阳盛，乃阴虚不能敛阳，阳浮于外。治则：滋阴以化阳，泻阳以保阴。即使阳虚用桂附，亦必少佐滋阴——"上焦阴分不虚而后可受参芪，下焦阴分不虚而后可受桂附"。

### 辨证五步决策树

```
问主症 → 定气机升降大方向
  │
  ├─ 气短/呼气困难/乏力/下坠感 →【下陷路径】
  │    │
  │    └─ 脉诊确认
  │         ├─ 右寸关沉迟微弱 → 大气下陷
  │         │    ├─ 兼心肺阳虚（身冷背紧）→ 回阳升陷汤
  │         │    ├─ 兼气郁（胸满胁胀）→ 理郁升陷汤
  │         │    ├─ 兼脾虚（小便不禁）→ 醒脾升陷汤
  │         │    └─ 单纯型 → 升陷汤
  │         │
  │         └─ 脉沉细无力但非寸关特甚 → 中气下陷（补中益气方向）
  │
  └─ 气上冲/呃逆/呕吐/头胀 →【上逆路径】
       │
       └─ 脉诊确认
            ├─ 右脉弦硬而长（大于左脉）→ 冲气上冲
            │    ├─ 兼肝火（左脉弦硬）→ 镇肝熄风汤
            │    ├─ 兼胃热（右脉洪滑）→ 白虎汤+赭石
            │    ├─ 阴分虚甚 → 参赭镇气汤
            │    ├─ 寒热夹杂 → 寒降汤/温降汤
            │    └─ 冲胃并逆 → 镇逆汤
            │
            └─ 脉非弦长、关尺沉 → 需进一步辨六经
```

### 脉诊三铁律

| 铁律 | 内容 | 原著依据 |
|------|------|----------|
| **有力无力定虚实** | 三部总按，有力→实证/热证，无力→虚证/气陷 | "有力无力，虚实立判；虚实一明，治则无差" |
| **辨真假有力** | 真有力有洪滑之象（波涛叠涌、累累如贯珠），弦直无起伏为假实（阴虚阳浮） | "若脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力" |
| **数脉主虚不主热** | 数而无力→大补元气，忌寒凉。数而有力方为真热 | "见数脉便用凉药，杀人无数" |

### 脉象-病机速查

| 脉象 | 部位 | 病机 | 治法方向 |
|------|------|------|----------|
| 弦细无力 | 左 | 肝血虚、大气下陷 | 升陷补气 |
| 弦硬有力长 | 左 | 肝肾阴亏、冲气上冲 | 镇逆+滋阴 |
| 弦硬有力长 | 右（大于左） | 冲气上冲、胃气不降 | 镇冲降胃 |
| 弦细无力 | 右 | 土为木伤、脾胃失运 | 健脾疏肝 |
| 沉迟微弱 | 右寸关 | 大气下陷 | 升陷汤 |
| 浮而无力 | 左右 | 失血过多、下焦不固 | 补血固涩 |
| 洪滑有力 | 右 | 阳明实热、胃气不降 | 白虎汤清降 |
| 尺脉无根 | 尺部 | 肝肾虚极、阴阳不维 | 急固脱 |

### 大气下陷 vs 气逆之喘——关键鉴别

| 鉴别点 | 大气下陷 | 气逆之喘 |
|--------|----------|----------|
| 呼吸困难在 | **呼气**难 | **吸气**难 |
| 肩息（喘时耸肩） | 必**不**肩息 | 剧者必然肩息 |
| 脉象 | 沉迟微弱（寸关尤甚） | 多数、浮滑、尺弱寸强 |

### 六经气化解读——对接仲景全库

张锡纯以气机升降视角重释六经，太阳经热力来自君相二火，"胸中大气"是枢纽——太阳表证本质是大气不足、卫外失固。

| 六经 | 气化解读 | 核心病机 | 治法 |
|------|----------|----------|------|
| 太阳 | 胸中大气+膀胱寒水 | 大气不足，卫外失固 | 桂枝汤补助大气 |
| 阳明 | 胃气不降+燥金 | 胃气当降不降，郁而化热 | 白虎/承气清降 |
| 少阳 | 胆+三焦，游部贯通 | 枢机不利 | 小柴胡调畅气机 |
| 太阴 | 脾气不升 | 中焦气化不足 | 理中汤温运 |
| 少阴 | 肾中水火 | 元阳不足或阴虚阳浮 | 四逆/黄连阿胶 |
| 厥阴 | 肝失疏泄 | 肝脏受邪，气机逆乱 | 乌梅丸调肝安冲 |
