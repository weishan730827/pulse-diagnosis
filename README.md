# 辨证施治辅助系统 v3.0

## 双轨辨证体系

本系统采用双轨并行架构，输出汇总后人工讨论定夺。

### 管线 A：知医邦量化 → 八家辨证

**输入端**：知医邦量化勾选表A（脉诊28维）+ 舌诊勾选表 + 体征

**八家辨证引擎**：输入量化数据后，分别交由八家独立辨证，输出各自辨证结果。

| 序号 | 医家 | 辨证体系 | rules 文件 |
|------|------|----------|-----------|
| 1 | 张仲景 | 六经辨证（伤寒+金匮原文） | zhang_zhongjing_rules.json |
| 2 | 胡希恕 | 六经八纲/方证对应 | hu_xishu_rules.json + fangzheng_db |
| 3 | 张锡纯 | 大气下陷/升陷汤一元化 | zhang_xichun_rules.json |
| 4 | 曹颖甫 | 经方实验/病汗药汗 | cao_yingfu_rules.json |
| 5 | 姚梅龄 | 脉证经方/独处藏奸 | yao_meiling_rules.json |
| 6 | 刘渡舟 | 方证相对/水证火证十论 | liu_duzhou_rules.json |
| 7 | 黄元御 | 四圣心源/六气解 | huang_yuanyu_rules.json + originals/ |
| 8 | 郑钦安 | 阴阳辨证/医理真传 | zheng_qinan_rules.json |

### 管线 B：陈建国独立脉诊

**输入端**：勾选表B（陈建国六步定向脉诊）

**独立辨证**：陈建国脉位签名体系 → 50方脉证经方匹配 → 独立输出

| 文件 | 用途 |
|------|------|
| 勾选表B_陈建国六步定向50方脉证经方.md | 脉诊输入端 |
| chen_jianguo_pulse_system.md | 六步定向辨证规则 |
| chen_jianguo_formula_signatures.json | 50方脉证签名数据库 |

### 汇总结论

管线 A（八家辨证结果）与管线 B（陈建国独立脉诊结果）汇总后，人工讨论定夺最终方案。

## 编码体系

所有输出编码统一三元标注：`[维度: 来源] 编码 = 含义`

严禁跨体系混用编码，严禁编造或推测编码，所有数据只从已核验的JSON原文输出。
