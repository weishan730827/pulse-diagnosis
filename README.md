# 辨证施治辅助系统 v2.0

整合姚梅龄脉学、张锡纯方证、胡希恕经方、郑钦安阴阳辨证、刘渡舟水证火证十论、曹颖甫经方实验、知医邦量化（脉诊+舌诊）七家辨证施治体系，提供多源综合辨证分析。

永远免费，人人清晰上手。

## 环境

Python 3.10+，无第三方依赖。

## 使用

```bash
git clone https://github.com/weishan730827/pulse-diagnosis.git
cd pulse-diagnosis
python3 bianzheng_shizhi_v2.py
```

## 命令

| 命令 | 功能 |
|------|------|
| `search 症状 / 脉象` | 综合辨证（五体系：脉象分析+知医邦量化+舌诊+方证匹配+郑钦安阴阳辨证+鉴别诊断） |
| `pulse 脉名` | 姚梅龄脉象诊断查询 |
| `zhiyibang 选项` | 知医邦28脉∈量化 |
| `tongue 选项` | 知医邦舌诊辨证 |
| `contraindication 方名` | 张锡纯用药禁忌查询 |
| `list-pulses` | 列出40种病脉 |
| `list-fangzheng` | 列出48条方证 |
| `list-formulas` | 列出28脉公式 |
| `list-tongue` | 列出舌诊选项 |

## 数据库

| 数据 | 规模 | 来源 |

## 编码体系

所有输出编码统一三元标注：`[维度: 来源] 编码 = 含义`

严禁跨体系混用编码，严禁编造或推测编码，所有数据只从已核验的JSON原文输出。
