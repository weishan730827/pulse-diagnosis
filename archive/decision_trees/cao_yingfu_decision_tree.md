# 曹颖甫辨证决策树（经方实验派）

> 来源：曹颖甫《经方实验录》(门人姜佐景编按，三卷92案)、《伤寒发微》、《金匮发微》
> 引擎：cao_yingfu_engine.py
> 路线：症状→方剂直接映射。方证对应，纯仲景方库。
> 学术特色：最纯粹的方证对应——勾选表中每一项症状直接对应一方，不经过六经或八纲中间层。

## 决策流程图

```
用户勾选症状
  ↓
[第1步] 方证直接映射
  勾选表中每项症状直接绑定一方
  如："恶寒发热无汗身痛" → 麻黄汤
      "发热汗出恶风" → 桂枝汤
  ↓
[第2步] 投票排名
  多个症状指向同一方→该方权重增加
  按命中症状数排序输出候选方
  ↓
[第3步] 输出推荐
  得分最高的方为首选，其余为备选
  全部经方，不做加减
```

## 核心学术思想

### 法遵仲景·恪守经方

> "经方派处方用药，谨守仲景之法。有是证，用是药，有是证，用是方。"
> "绝不经易加减，加减一味，定有依据。"

| 原则 | 内容 |
|------|------|
| 方证对应 | 症状直接映射方剂，不走六经八纲中间层 |
| 恪守原方 | 不随意加减药味，加减必有经文依据 |
| 心小胆大 | 早年桂枝芍药仅三钱（原方三两的十分之一），晚年三五钱。随经验递增而加大剂量 |
| 败案反思 | 无效案例如实记录并分析原因——丸药质量问题、煎服方法不当等 |

### 不拘病名·唯证是辨

> "所谓中风、伤寒、温病、太阳病……不过是疾病之代名，非疾病之真名。"
> "苟病者所犯为大承气汤证，则投以大承气汤，可也。不必问其为伤寒、为温病。"

### 病汗与药汗（特色辨证）

| 类型 | 特征 | 临床意义 |
|------|------|----------|
| 病汗 | 疾病本身的汗出，多带凉意 | 不可见汗止汗——需辨证求本 |
| 药汗 | 服发汗药后遍身微汗出，多带热意 | 治疗目的——'遍身漐漐微似有汗者益佳' |

**关键推论**：
- 有汗出者仍可用发汗药——"病汗"不碍"药汗"
- 发汗以"遍身微汗"为度，不可令如水流漓

## 92案统计特征

| 特征 | 数据 |
|------|------|
| 总医案 | 92案（三卷） |
| 仲景方使用率 | >90% |
| 最常用方 | 桂枝汤、麻黄汤、大承气汤、小柴胡汤 |
| 剂量特征 | 早年极谨慎（原方1/10），晚年加大（三五钱） |
| 剂型用法 | 强调煎服法——桂枝汤须啜热稀粥，麻黄汤先煮去上沫 |
| 治学态度 | 无效案例如实记录，分析原因（丸药质量、煎法错误等） |

## 辨证特色总结

| 维度 | 曹颖甫 |
|------|--------|
| 辨证入口 | 症状直接映射方剂（最简路径） |
| 中间层 | 无——不经过六经、八纲 |
| 方剂池 | 纯仲景方 |
| 药味加减 | 极少，加减必有经文依据 |
| 剂量哲学 | 从谨慎到大胆的实战演进 |
| 学术态度 | 败案公开、自我反思 |
