## 核心总纲

> **实则左升右降，虚则右升左降。**

| 四象 | 脉诊 | 治法 | 代表方 |
|------|------|------|--------|
| 阴盛 | 左手太过 | 辛温升法（汗法） | 麻黄汤、小柴胡汤 |
| 阳虚 | 右手不及 | 甘温升法（补阳） | 四逆汤、理中汤 |
| 阳盛 | 右手太过 | 苦寒降法（下法） | 大承气汤、大黄黄连泻心汤 |
| 阴虚 | 左手不及 | 甘寒降法（滋阴） | 百合地黄汤、麦门冬汤 |

## 决策流程图

```
触摸18格脉（左右×寸关尺×浮中沉）
  ↓

  感受脉动力量（对应阳气盛衰）+ 脉管粗细（对应阴津血盛衰）
  ├── 总体太过：脉突出、容易摸到
  └── 总体不及：脉沉细无力、不容易摸到
  ↓

    └── 右脉更为太过 → 苦寒降法（阳盛）
  总体不及：
    ├── 左脉不及/左脉相对太过 → 甘寒降法（阴虚）

  ↓
```

## 理论基石

### 阴阳盛衰关系

| 经典出处 | 原文 | 含义 |
|----------|------|------|
| 素问 | 阳胜则阴病，阴胜则阳病 | 阴阳盛衰互为因果 |
| 伤寒论 | 阳盛阴虚，汗之则死，下之则愈。阳虚阴盛，汗之则愈，下之则死 | 治法方向定生死 |
| 灵枢 | 阴盛而阳虚，先补其阳后泻其阴；阴虚而阳盛，先补其阴后泻其阳 | 先后次第 |

### 因果链条
```
阳盛 → 热 → 伤津 → 阴虚（阳盛常伴阴虚）→ 皆用降法
阴盛 → 寒 → 伤阳 → 阳虚（阳虚常伴阴盛）→ 皆用升法
```

## 升降治法详解

> "汗之、下之"非特指汗法下法，而是治疗方向的约略之辞——汗法代表向上向外的"升法"，下法代表向下向内的"降法"。

### 四气五味与升降

| 属性 | 方向 | 药物 |
|------|------|------|
| 温热（四气） | 升法 | 阳药 |
| 寒凉（四气） | 降法 | 阴药 |
| 辛甘淡（五味） | 升法 | 阳药 |
| 酸苦咸（五味） | 降法 | 阴药 |

### 八法映射

| 方向 | 包含治法 |
|------|----------|
| 升法 | 汗法、吐法、温法、补法（甘味益气）、和法（辛温为主如小柴胡汤） |
| 降法 | 下法、清法、消法、补法（寒性滋阴如芍药甘草汤）、和法（苦寒为主如乌梅丸） |

### 疑难药物处理
气与味阴阳属性相反时，以药物作用的实际结果为准：
- 麻黄味苦性温 → 质地轻、气厚味薄、动性强 → 阳药升法
- 柴胡味苦 → 质地疏松、气厚味轻、推陈致新 → 升法（后世总结为"升阳"）

### 复方方向判定
1. 依君药或用量最大者定方向
2. 依仲景原文明示（如"与大柴胡汤下之则愈"→降法）

| 方剂 | 判定依据 | 方向 |
|------|----------|------|
| 白虎汤 | 君药石膏甘寒质重一斤 | 降法 |
| 五苓散 | 泽泻用量最大，总体降法药物占主体 | 降法（利小便） |
| 小柴胡汤 | 原文"上焦得通…身濈然汗出而解" | 升法 |
| 大柴胡汤 | 原文"下之则愈" | 降法 |

## 临床操作步骤

### 第1步：确定总体太过/不及
- 全面诊查双侧寸口脉（左右手、寸关尺、浮中沉，必须包括沉位）
- 感受脉动力量（对应阳气盛衰）和脉管粗细（对应阴津血盛衰）

| 术语 | 定义 |
|------|------|
| 太过 | 脉突出、容易摸到 → 脉实证实 |
| 不及 | 脉沉细无力、不容易摸到 → 脉虚证虚 |
| 相对太过 | 总体不及前提下某一部相对突出——两者治法方向几乎相反 |

### 第2步：定位最强脉位
- 总体太过 → 左盛升法，右盛降法
- 总体不及 → 左虚降法，右虚升法

### 第3步：定具体方证
- 水饮+左脉太过 → 升法（厚朴麻黄汤、苓桂术甘汤）
- 水饮+右脉太过 → 降法（泽漆汤、枳术汤）
- 水饮+右脉不及 → 温阳升法（真武汤）
- 水饮+左脉不及 → 滋阴降法（肾气丸）

## 四象口诀

```
阴津血为阴，阳气为阳。
左脉诊阴盛衰，右脉诊阳盛衰。
辛温汗法为升，苦寒下法为降。
实则左升右降，虚则右升左降。
```

## 脉诊历史探索（否定路径）

| 被否定的方法 | 问题 |
|-------------|------|
| 从脉象分阴阳（浮数=阳，沉弱=阴） | 阴盛无法表现为"盛" |
| 从尺寸分阴阳（寸阳尺阴） | 桂枝汤寸浮尺弱→阳盛阴虚？实则汗法升法 |
| 从浮沉分阴阳（浮=阳=汗，沉=阴=下） | 阳盛应用下法，浮却对应汗法 |
| 从气血营卫分阴阳 | 与阴阳盛衰诊断大法方向完全不同 |
| 从脉之去来分阴阳 | 理论符合但难以体会传承 |

> 最终结论：以左右手脉分阴阳，以太过不及定盛衰——临床反复验证与阴阳盛衰严丝合缝。
