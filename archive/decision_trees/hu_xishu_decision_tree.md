# 胡希恕辨证决策树

> 来源：《伤寒论讲座》《金匮要略讲座》+ hu_xishu_rules.json v2.0
> 引擎：hu_xishu_engine.py
> 路线：勾选表 → 八纲归类 → 六经锁定 → 方证输出

## 决策流程图

```
用户勾选症状
  ↓
[第1步] 阴阳判定
  ├── 阳证（发热恶寒/但热不寒，脉浮/数/大/滑/实有力，精神不衰）
  └── 阴证（无热恶寒/但寒不热，脉沉/微/细/弱无力，但欲寐）
  ↓
[第2步] 表里半判定
  ├── 表证：脉浮，恶寒（必见），头项强痛，身痛
  ├── 里证：脉沉，不恶寒，二便异常，腹证
  └── 半表半里：胸胁苦满，往来寒热，口苦咽干目眩
  ↓
[第3步] 六经锁定（阴阳×表里半 → 六经）
  ├── 阳+表 → 太阳病
  ├── 阴+表 → 少阴病
  ├── 阳+里 → 阳明病
  ├── 阴+里 → 太阴病
  ├── 阳+半 → 少阳病
  └── 阴+半 → 厥阴病
  ↓
[第4步] 寒热虚实细化
  ├── 热证：口渴、烦躁、面赤、舌红、脉数
  ├── 寒证：口不渴或喜热饮、手足冷、下利清谷、脉迟
  ├── 实证：脉实有力，腹痛拒按，大便不通
  └── 虚证：脉虚无力，腹痛喜按，下利清谷
  ↓
[第5步] 方证对应
  六经锁定 + 证候群 → 精确方剂
```

## 核心学术原则

| 原则 | 内容 |
|------|------|
| 学术定位 | 中医辨证施治以张仲景为代表，根据一般规律找出的方法，治一切疾病 |
| 与内经关系 | 《伤寒论》与《内经》毫无关系，取《汤液经》方证体系自成一家 |
| 辨证口诀 | 先辨六经，继辨八纲，最后方证对应。有是证用是方 |
| 津液保护 | 发汗、攻下均伤津液。津液亡失可致直视、失溲等坏病 |

## 六经体系详解

### 太阳病
- **病位**：表，阳
- **提纲**：脉浮，头项强痛而恶寒
- **核心病机**：机体欲通过发汗解除疾病而未能出汗，正邪交争于表

| 子型 | 主症 | 主脉 | 代表方 |
|------|------|------|--------|
| 太阳伤寒 | 恶寒、无汗、体痛、呕逆 | 浮紧 | 麻黄汤 |
| 太阳中风 | 发热、汗出、恶风 | 浮缓/浮弱 | 桂枝汤 |
| 大青龙汤证 | 恶寒发热+无汗+烦躁 | 浮紧 | 大青龙汤 |

### 少阴病
- **病位**：表，阴
- **核心鉴别**：无热恶寒者为少阴
- **治疗原则**：温阳解表，不可单纯发汗伤阳
- **代表方**：麻黄附子甘草汤、麻黄附子细辛汤

### 阳明病
- **病位**：里，阳
- **主症**：身热汗出，不恶寒反恶热
- **治疗原则**：清（白虎）、下（承气），禁发汗

| 方证 | 必备条件 | 方 |
|------|----------|----|
| 白虎汤证 | 大热+大汗+大渴 | 白虎汤 |
| 承气汤证 | 潮热+谵语+腹满硬痛+大便不通 | 承气汤类 |

### 太阴病
- **病位**：里，阴
- **主症**：腹满而吐、食不下、自利益甚、时腹自痛
- **代表方**：理中汤、四逆汤

### 少阳病
- **病位**：半表半里，阳
- **主症**：往来寒热、胸胁苦满、默默不欲饮食、心烦喜呕、口苦咽干目眩
- **主脉**：弦细、弦数

| 方证 | 必备条件 | 方 |
|------|----------|----|
| 小柴胡汤证 | 胸胁苦满+往来寒热+心烦喜呕+口苦 | 小柴胡汤 |
| 大柴胡汤证 | 小柴胡汤证+呕不止+心下急+便秘 | 大柴胡汤 |

### 厥阴病
- **病位**：半表半里，阴
- **主症**：消渴、气上撞心、心中疼热、饥而不欲食
- **代表方**：乌梅丸

## 六经皆有表证（关键规则）

| 六经 | 表证特征 | 对应方 |
|------|----------|--------|
| 太阳 | 脉浮、恶寒发热 | 桂枝汤/麻黄汤类 |
| 少阴 | 脉沉细、但恶寒、无热 | 麻黄附子甘草汤类 |
| 阳明 | 阳明病兼太阳未罢 | 先解表后攻里 |
| 太阴 | 太阴病兼表证 | 桂枝人参汤 |
| 少阳 | 少阳兼太阳 | 柴胡桂枝汤 |
| 厥阴 | 厥阴病兼表 | 先温里后攻表 |

## 八纲辨证框架

### 表里判定
| 分型 | 特征 |
|------|------|
| 表证 | 脉浮，恶寒（必见），头项强痛，身痛。正气欲从表解邪 |
| 里证 | 脉沉，不恶寒（恶热或但寒不热），二便异常，腹证 |
| 半表半里 | 胸胁苦满，往来寒热，口苦咽干目眩。不在表不在里 |

### 阴阳判定
| 分型 | 特征 |
|------|------|
| 阳证 | 发热恶寒/但热不寒，脉浮/数/大/滑/实有力，精神不衰 |
| 阴证 | 无热恶寒/但寒不热，脉沉/微/细/弱无力，但欲寐/精神萎靡 |

### 寒热虚实
| 分型 | 特征 | 方向 |
|------|------|------|
| 热证 | 口渴、烦躁、面赤、舌红、脉数 | 清法 |
| 寒证 | 口不渴或喜热饮、手足冷、下利清谷、脉迟 | 温法 |
| 实证 | 脉实有力，腹痛拒按，大便不通，邪气盛 | 攻法 |
| 虚证 | 脉虚无力，腹痛喜按，下利清谷，正气虚 | 补法 |

## 胡希恕脉诊体系

> 三大观察角度：脉之动（搏动状态）、脉之体（血管形态）、血之行（血流状态）

### 脉之动：频率、节律、力度、浮沉

| 维度 | 含义 | 胡希恕解读 |
|------|------|-----------|
| 频率（迟数） | 一息几至 | 迟=寒，数=热 |
| 节律（结代促） | 有无歇止 | 结=脉跳跳一止，较轻；代=中止良久再来，极坏 |
| 力度（虚实） | 有力无力 | 实证=有力，虚证=无力 |
| 浮沉（脉位） | 浅深 | 浮=病在表/正气趋表，沉=病在里 |

### 脉之体：血管状态（两对比轴）

| 对比轴 | 观察角度 | 脉象 | 含义 |
|--------|----------|------|------|
| 紧缓轴 | 横向紧束感 | 紧脉 | 体液充实于表，血管充血严重，太阳伤寒无汗 |
| 紧缓轴 | 横向紧束感 | 缓脉 | 体液部分丧失（已出汗），脉道弛张，非指脉率慢 |
| 弦弱轴 | 纵向端直度 | 弦脉 | 上下端直如琴弦，正气不虚，病欲传少阳或水饮 |
| 弦弱轴 | 纵向端直度 | 弱脉 | 弦上得不紧，软弱无力，津血不足 |

### 血之行：血流运行

| 脉象 | 含义 |
|------|------|
| 滑脉 | 热、孕 |
| 涩脉 | 血不足 |
| 动脉 | 摇摆不稳，正气将竭之危象 |

### 促脉的特殊定义

> 胡希恕明确反对王叔和"数中一止谓之促"。胡希恕促脉=迫于上、迫于外，寸脉浮关尺沉。表未解而气上冲之象。

## 传变判断

| 判断 | 依据 |
|------|------|
| 不传 | 脉静者不传 |
| 传变 | 脉数急者为传 |
| 传少阳 | 颇欲吐 |
| 传阳明 | 躁烦 |
