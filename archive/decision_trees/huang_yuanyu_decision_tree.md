# 黄元御辨证决策树（一气周流）

> 来源：《四圣心源》(1753)、《伤寒悬解》(1748)、《金匮悬解》(1754)、《长沙药解》(1753)
> 引擎：huang_yuanyu_engine.py
> 路线：中气为轴→四维为轮→升降方向→方证锁定。全仲景方库。
> 学术定位：填补系统'气机升降'维度空白——此前八家均未从一气周流角度进行系统辨证。

## 核心总纲

> **土枢四象，一气周流。**
> 阴升阳降，中气斡旋。四维之病，悉因于中气。

## 决策流程图

```
[第1步] 判中气状态
  中气者阴阳升降之枢轴（脾为己土主升，胃为戊土主降）
  ├── 中气旺：胃降善纳，脾升善磨 → 无病
  └── 中气衰：升降窒塞，四维皆病
  ↓
[第2步] 定升降偏颇（五行模式匹配）
  ├── 脾虚为主：中气不足，脾不升清 → 左路升发无力 → 肝郁/清阳不升/湿困
  ├── 胃逆为主：胃气不降，浊阴上逆 → 右路肃降不足 → 肺气不降/心火不下/上热
  ├── 脾虚胃逆：中气溃败，升降俱废 → 清浊相干 → 上热下寒/痰湿内停
  ├── 食积停滞：饮食积滞，中轴受阻 → 腐浊内生
  └── 清阳不升：脾虚下陷的延伸 → 清气不升 → 头晕/疲倦/泄泻
  ↓
[第3步] 方证锁定（恢复轴心→调畅四维）
  先复中气之斡旋，再调四象之升降
  方剂全部对接仲景基座
```

## 理论基石

### 一气周流模型

```
              （上）心火
              ↗     ↘
          肝木        肺金
          ↗             ↘
    脾（己土左旋）  胃（戊土右降）
            ↖     ↙
             中 气
           （太极·祖气）
              ↓
             肾水（下）
```

| 要素 | 含义 |
|------|------|
| 中气 | 阴阳升降之枢轴，即脾胃。中气旺则四象各得其位 |
| 己土（脾） | 左旋而升，带动肾肝亦升——水木不郁 |
| 戊土（胃） | 右转而降，带动心肺亦降——金火不滞 |
| 四象 | 水(肾)、火(心)、金(肺)、木(肝)，即阴阳之升降的不同阶段 |
| 祖气 | 人身之太极，内含阴阳，阴阳之间即中气 |

### 病机推导

```
中气衰
  ├── 脾不升 → 肾肝不升
  │   ├── 肾水下寒 → 精病（遗泄不秘）
  │   └── 肝木左郁 → 血病（凝瘀不流）
  └── 胃不降 → 心肺不降
      ├── 心火上炎 → 神病（惊怯不宁）
      └── 肺金右滞 → 气病（痞塞不宣）
```

## 五大辨证模式

| 模式 | 病机链 | 首选方 | 四维调方 |
|------|--------|--------|----------|
| **脾虚为主** | 脾虚→左路升发无力→肝郁/清阳不升/湿困 | 苓桂术甘汤（健脾利湿，恢复轴心） | 肝：四逆散；清阳不升：理中汤；水湿：五苓散；下寒：真武汤 |
| **胃逆为主** | 胃逆→右路肃降不足→肺气不降/心火不下/上热 | 旋覆代赭汤（降逆和胃，恢复轴心） | 肺：麻杏甘石汤；心：黄连阿胶汤；胃火：白虎汤；食积：生姜泻心汤 |
| **脾虚胃逆** | 中气溃败→清浊相干→上热下寒→痰湿内停 | 半夏泻心汤（辛开苦降，恢复斡旋） | 上热下寒：乌梅丸；痰湿：小半夏加茯苓汤；升降不通：半夏厚朴汤 |
| **食积停滞** | 食积→中气不运→升降受阻→腐浊内生 | 生姜泻心汤（消痞和胃） | 胃逆：小承气汤；湿热：葛根黄芩黄连汤 |
| **清阳不升** | 脾虚下陷→清气不升→头晕/疲倦/泄泻 | 理中汤（温中升阳） | 下利：葛根汤；湿困：平胃散→术附汤 |

## 脏腑气机升降表

| 脏 | 方向 | 腑 | 方向 |
|----|------|----|------|
| 肝（乙木） | ↑ 升发 | 胆（甲木） | ↓ 降泄 |
| 心（丁火） | ↑ 温煦（君火） | 小肠（丙火） | ↓ 分清泌浊 |
| 脾（己土） | ↑ 升清 | 胃（戊土） | ↓ 降浊 |
| 肺（辛金） | ↓ 肃降 | 大肠（庚金） | ↓ 传导 |
| 肾（癸水） | ↑ 气化 | 膀胱（壬水） | ↓ 排泄 |

## 黄元御与本案辨证体系的关联

- 张仲景（本经）定义了"八法"——汗吐下和温清消补——但未从气机升降角度统一解释
- 张锡纯从"大气下陷/肝阳上逆"切入升降，但未建立系统框架
- 胡希恕以六经八纲辨证，不涉及气机模型
- 陈建国"实则左升右降"虽涉及左右升降，但属治法方向而非生理模型
- **黄元御是唯一从"气"的一元论出发，将脏腑、四象、五行、升降统一为"一气周流"的医家**
