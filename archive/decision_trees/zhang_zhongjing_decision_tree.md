## 决策流程图

```
[第1步] 六经提纲锁定（八纲总判）
  据《伤寒论》六经提纲证分类：
  ├── 太阳：脉浮，头项强痛而恶寒 → 表阳证 → 解表发汗
  ├── 阳明：胃家实（身热汗出不恶寒反恶热） → 里阳证 → 清下两法
  ├── 少阳：口苦咽干目眩，往来寒热胸胁苦满 → 半表半里阳证 → 和法
  ├── 太阴：腹满而吐，食不下，自利益甚 → 里阴证 → 温法
  ├── 少阴：脉微细，但欲寐 → 表阴证 → 寒化四逆/热化黄连阿胶
  └── 厥阴：消渴气上撞心心中疼热 → 半表半里阴证 → 乌梅丸
  ↓
[第2步] 变证兼证细化
  太阳变证（已离表）、坏病兼证、合病/并病判定
  ↓
[第3步] 方证索引
  六经归属 + 变证 → 原文方剂（113方）
```

## 总纲

> **"观其脉证，知犯何逆，随证治之。"**（第16条）
> 不拘病名，唯脉证是辨。此为仲景脉法之根。

## 治则八法

| 法 | 原文依据 | 核心方剂 | 关键禁忌 |
|----|----------|----------|----------|
| 汗法 | 脉浮者病在表，可发汗 | 麻黄汤、桂枝汤、葛根汤、大小青龙汤 | 咽喉干燥者、淋家、疮家、衄家、亡血家、汗家不可发汗 |
| 吐法 | 胸中痞硬，气上冲喉咽不得息→当吐之 | 瓜蒂散 | 亡血虚家不可与 |
| 下法 | 阳明病谵语潮热，胃中燥屎→大承气汤 | 大/小/调胃承气汤、桃核承气汤、抵当汤 | 呕多不可攻、心下硬满不可攻、面赤不可攻 |
| 和法 | 往来寒热胸胁苦满→小柴胡汤 | 小/大柴胡汤、柴胡桂枝汤 | 脉迟浮弱恶风寒→柴胡汤不中与 |
| 温法 | 自利不渴属太阴→当温之 | 四逆汤、通脉四逆汤、白通汤、理中丸 | 少阴脉细沉数不可发汗 |
| 清法 | 伤寒脉滑而厥→里有热→白虎汤 | 白虎汤、竹叶石膏汤、栀子豉汤、黄连阿胶汤 | 表不解不可与白虎汤 |
| 消法 | 少腹满应小便不利今反利→有血→抵当丸 | 抵当汤/丸、桃核承气汤、十枣汤、大小陷胸汤 | 针对瘀血、蓄水、痰饮、结胸 |
| 补法 | 伤寒心中悸而烦→小建中汤 | 小建中汤、炙甘草汤、芍药甘草汤、肾气丸 | - |

| 总纲 | 观其脉证，知犯何逆，随证治之 | 不拘病名，唯脉证是辨 |

## 审察病机五条

| 条目 | 核心规则 |
|------|----------|
| 辨证总纲 | 先辨阴阳→再定六经→继查变证→后方证相应 |
| 表里先后 | 表里同病：本发汗而反下之此为逆；先下之治不为逆 |
| 寒热真假 | 病人身大热反欲得衣=热在皮肤寒在骨髓（真寒假热）；反之亦然 |
| 虚实转化 | 实则阳明（阳明多实），虚则太阴（太阴多虚） |
| 新久先后 | 新病+痼疾并存→先治新病后治痼疾 |
