> 学术特色：阴阳二分法，"辨阳虚之关键不在热象有无，而在阳气虚损的本质"

## 决策流程图

```
症状输入
  ↓
[第1步] 核心三问定阴阳
  一问渴饮、二问舌象、三问脉象
  ├── 不渴/喜热饮 + 舌淡白润 + 脉无力 → 阴证（阳虚）
  └── 渴喜冷饮 + 舌红干黄 + 脉有力 → 阳证（阴虚阳盛）
  ↓
[第2步] 扩展辨证细化
  在阴阳大方向下细化方证选择
  ↓
[第3步] 方剂推荐
  阴证→温阳方（四逆汤、理中汤等）
  阳证→清热/泻下方（白虎汤、承气汤等）
  自创方（潜阳丹、封髓丹等）→附录标注
```

## 核心学术思想

### 阴阳总纲

> "医学一途，不难于用药，而难于识症。亦不难于识症，而难于识阴阳。"
> "万病不出阴阳二字。"

### 阳虚辨识（十二症）

| 维度 | 阳虚特征 |
|------|----------|
| 面色 | 唇口青白无神 |
| 神态 | 目瞑倦卧，声低息短，少气懒言 |
| 体感 | 身重畏寒，自汗肢冷 |
| 口中 | 口吐清水，满口津液，不思水饮，即饮喜热汤 |
| 舌象 | 舌青滑、黑润青白色、淡黄润滑色 |
| 二便 | 二便自利 |
| 脉象 | 脉浮空、细微无力 |
| 其他 | 爪甲青、腹痛囊缩 |

### 阴虚辨识（九症）

| 维度 | 阴虚特征 |
|------|----------|
| 面色 | 面目唇口红色 |
| 神态 | 精神不倦，张目不眠，声音响亮，口臭气粗 |
| 体感 | 身轻恶热 |
| 口中 | 口渴饮冷 |
| 舌象 | 舌苔干黄或黑黄，全无津液，芒刺满口 |
| 二便 | 二便不利 |
| 脉象 | 六脉长大有力 |
| 其他 | 烦躁谵语、潮热盗汗、干咳无痰 |

### 阳虚阴盛核心病机（火神派精髓）

> "阳气不足，稍有阻滞，百病丛生。"

| 病机 | 说明 |
|------|------|
| 核心公式 | 阳虚为本，阴盛为标 |
| 阴盛产物 | 阳不化阴→湿浊内生、水饮停聚、瘀血凝结 |
| 衍生症状 | 水肿、痰饮、脚气、囊肿、积液、泄泻、癥瘕 |
| 上火假象 | 外虽现一切火症（口疮、咽痛、牙痛），近似实火——实为阳虚真寒 |

### 真阳判别法

> "凡阳虚之人，阴气自然必盛。外虽现一切火症，近似实火，俱当以此法辨之，万无一失。"

| 关键 | 内容 |
|------|------|
| 判断基准 | 不在热象有无，而在阳气虚损的本质 |
| 必见症状 | 身重恶寒、目瞑嗜卧、声低息短、少气懒言、渴喜热饮 |
| 火象本质 | 火症为假热——阳虚真寒，虚阳上浮 |

## 辨证要点

| 对比 | 阴证（阳虚） | 阳证（阴虚阳盛） |
|------|-------------|-----------------|
| 渴饮 | 不渴或喜热饮 | 口渴喜冷饮 |
| 舌象 | 淡白润滑 | 红干黄燥 |
| 脉象 | 细微无力/浮空 | 长大有力 |
| 神态 | 倦卧少气 | 不眠响亮 |
| 体温 | 畏寒肢冷 | 恶热 |
| 治法 | 温阳（四逆汤、理中汤、白通汤） | 清热泻下/滋阴（白虎汤、承气汤、黄连阿胶汤） |
