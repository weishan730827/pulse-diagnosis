# 知医邦量化表 · 脉诊勾选表

> 公式来源：知医邦（李华渊）公开 AI 脉诊量表
> 目标：将"指下感觉"转化为可复现的物理维度描述

---

## 把脉指法

中指定关（桡骨茎突最高处）→ 食指寸 → 无名指尺，三指并拢，依次轻按（浮）、中按、重按（沉）。

---

## 总体（搭双手，不分部）

| 维度 | 勾选 |
|------|------|
| **C 频率** | □ CA 慢(<60) □ CB 稍慢(60-69) □ CE 不快不慢(70-89) □ CC 快(90-119) □ CD 极快(≥120) |
| **D 节律** | □ DA 不齐(杂乱无章) □ DB 停顿(有卡顿歇止) □ DC 均匀(平稳无卡停) |

## 左手

### 左寸

| 维度 | 选项 |
|------|------|
| A 深度 | □ AA浅(触皮即得，重按反而减弱) □ AB深(轻按不得，重按始得) □ AC极深(重压着骨乃得) □ AD不深不浅 |
| B 长度 | □ BA短(三指挤拢，食/无名指无脉) □ BB长(三指分开仍能感到) □ BC不长不短 |
| E 宽度 | □ EA宽(如条带，切向宽大) □ EB细(如丝线，切向细小) □ EC中 |
| F 紧度 | □ FA紧(紧绷，左右弹指，如绳索) □ FB空(中空，稍按即瘪，如葱管) □ FC中 |
| G 硬度 | □ GA硬(发硬挺然，如按琴弦) □ GB软(柔软若有若无，如触水面) □ GC中 |
| H 流利度 | □ HA滑利(血流快速、匀速、滑利) □ HB艰涩(血流慢、变速、艰涩) □ HC中 |
| I 力度 | □ IA无力(跳动幅度小，应指无力) □ IB有力(跳动幅度大，应指有力) □ IC均衡 |
| J 坡度 | □ JA势缓(来如登山，去如山崩) □ JB势急(来如山倒，去如抽丝) □ JC从容和缓 |

## 知医邦28脉公式速查

| 脉名 | 公式 | 对应用途 |
|------|------|----------|
| 浮 | AA | 表证，虚阳外越 |
| 沉 | AB | 里证 |
| 伏 | AC | 邪闭、厥证 |
| 迟 | CA | 寒证 |
| 数 | CC | 热证 |
| 疾 | CD | 阳极阴竭 |
| 滑 | {HA}+IB+(CC) | 痰、食、孕 |
| 涩 | {HB}+JA+EB+(DA)+(BA) | 气滞血瘀 |
| 虚 | {IA}+GB+(EA)+(AA) | 气血两虚 |
| 实 | {IB}+GA+(EA)+(BB) | 邪气有余 |
| 弦 | {GA}+BB+(IB) | 肝胆病、痛、痰饮 |
| 细 | {EB}+IA+(AB) | 气血两虚、湿 |
| 濡 | {GB}+AA+(EB)+(IA) | 虚证、湿证 |
| 弱 | GB+AB+(EB)+(IA) | 气血不足 |
| 洪 | {EA}+JB+IB+(AA)+(CC) | 热盛 |
| 紧 | {FA}+IB+(BB)+(JB)+(CC) | 寒、痛、宿食 |
