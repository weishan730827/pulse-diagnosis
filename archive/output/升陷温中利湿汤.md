# 升陷温中利湿汤

## 方源

升陷汤（张锡纯）+ 苓桂术甘汤（伤寒论/金匮要略）+ 补中益气汤（李东垣）三方化裁

## 方药组成

| 药物 | 用量 | 炮制/备注 |
|------|------|----------|
| 生黄芪 | 60g | 君药，补气升陷 |
| 知母 | 20g | 佐制黄芪燥热 |
| 柴胡 | 6g | 引经升提 |

| 升麻 | 6g | 引经升提 |

共16味。

## 煎服法

龙骨、牡蛎先煎30分钟，余药后下。
头煎40分钟，二煎30分钟。
混合分早晚温服。

## 适应证

大气下陷（升陷汤证）+ 太阴脾虚（苓桂术甘汤证）+ 脾胃清气下陷（补中益气汤证）

主症：畏寒怕冷，心慌气短，小便不禁，食欲不振，大便不成型，气顶感，脚气手癣脚癣，眼干，喜热饮。
舌诊：舌淡粉，左侧隆起，横裂纹多，舌尖点状凸起，水滑苔白。
脉诊：左寸中取濡，尺沉细，右寸弱。

## 方义

- 升陷汤升胸中大气（黄芪+知母+升柴桔）
- 苓桂术甘汤降水饮上冲（桂枝+茯苓+白术+甘草）
- 补中益气汤升脾胃清气（黄芪+人参+白术+当归+升柴+甘草）
- 山萸肉+龙牡固涩防脱
- 干姜+桂枝温阳（回阳升陷汤法）
- 萆薢→茯苓利湿浊
- 骨碎补+山药补肾
- 当归养血治眼干

## 辨证体系对照

| 症状 | 对应药物 | 体系 |
|------|----------|------|
| 气短、右寸弱 | 黄芪+桂枝 | 张锡纯升陷 |

| 心慌 | 桂枝温通心阳 | 回阳升陷汤 |
| 小便不禁 | 龙牡+萸肉+骨碎补 | 醒脾升陷汤固涩 |
| 食欲不振大便不成型 | 白术+干姜+茯苓 | 太阴脾虚温中 |
| 气顶感 | 黄芪升提+桂枝平冲 | 理郁升陷+苓桂术甘 |
| 脚气手癣脚癣 | 茯苓利湿 | 醒脾升陷汤法 |
| 眼干 | 当归 | 回阳升陷汤养血 |

| 水滑苔白 | 白术+干姜+茯苓 | 温化+渗湿 |

## 禁忌

1. 不可误作气逆用降气法（忌代赭石、旋覆花等）
2. 不可加肉桂（龙牡已固涩，且升陷体系不用肉桂）
3. 不可加麦冬（本案寒湿背景，麦冬甘寒不宜）
