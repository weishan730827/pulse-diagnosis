# 张锡纯脉诊辨治系统详解

> 本文从《医学衷中参西录》全文中逐条提取张锡纯脉诊原文，按他自身的理论框架组织。
> 所有辨证结论和方剂对应均直接出自原著，不添加外部脉学分类。

---

## 一、总纲：脉理三辨

### 1.1 辨有力之真假：脉之核心法则

张锡纯对于"脉有力"提出了整个中医脉学史上极为独特的真假之辨：

> **"脉之有力，有真有假。凡脉之真有力者，当于敦浓和缓中见之，此脾胃之气壮旺，能包括诸脏也。"**

他将有力分为三类假脉和一类真脉：

| 脉象 | 辨证 | 原文 |
|------|------|------|
| 洪而有力 | 外感之实热 | "其余若脉象洪而有力，多系外感之实热。" |
| 滑而有力 | 中焦之热痰 | "若滑而有力，多系中焦之热痰。" |
| 弦而有力 | 肝经偏盛，尤为有病之脉 | "若弦而有力，多系肝经之偏盛，尤为有病之脉。" |
| 敦浓和缓中有力 | 脾胃之气壮旺（真有力） | "凡脉之真有力者，当于敦浓和缓中见之。" |

**关键鉴别**：真有力必有洪滑之象——

> **"脉之真有力者，皆有洪滑之象。洪者如波涛叠涌，势作起伏；滑者指下滑润，累累如贯珠。此脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力之象。"**

真有力 = 有起伏（洪）+ 有滑润（滑），二者缺一即假有力。

### 1.2 辨脉之数与不数：气分决定论

> **"理气药多于补气药，则脉即加数；补气药多于理气药，则脉即渐缓。是知脉之数与不数，固视乎血分之盈亏，实尤兼视乎气分之强弱。"**

核心论点：**脉数不纯属阴虚，必兼气分虚弱。**

> **"世俗医者，遇脉数之证，大抵责之阴虚血涸。不知元气虚极莫支者，其脉可至极数。"**

实证：脉数至七八至而治愈的案例多处。张锡纯明确反对叔和脉法"数至七八至为不治"之论——此论是建立在"阴虚血涸"的旧框架上，张锡纯加入"气分虚弱"维度后，很多七八至可治：

> **"始知叔和脉法谓数至七八至为不治之脉者，非确论也。"**

### 1.3 辨脉之迟数与大气关系

> **"脉之迟数，恒关于人身之热力，热力过盛则脉数，热力微弱则脉迟，此定理也。今此证虽有伏气化热，因大气下陷而脉仍迟。"**

大气下陷可覆盖热力对脉数的影响——即使体内有伏热，大气下陷时脉仍可迟。

> **"大气下陷之脉大抵皆迟，非因寒凉而迟也。"**

## 二、脉象形态学：张锡纯特有的脉象分类

### 2.1 正脉标准

> **"和缓者脾胃之正脉，弦长者肝胆之正脉。"**

- **和缓** = 脾土之气濡养六部，是健康基准脉。有胃气 = 敦浓和缓。
- **弦长** = 肝胆正常脉象。但弦长本身不是病脉，关键看是否失去和缓。

> **"六部之脉皆有和缓，乃为正象。"**

### 2.2 弦直 = 脉无起伏

> **"脉无起伏为弦，弦而有力，即紧脉也。若但弦，则为寒矣。"**

这是张锡纯的一个关键定义：**脉的弦 = 无起伏**。正常脉应如"波涛叠涌"有波动性，失去波动、呈直线形态即为弦。

> **"脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力之象。"**

弦直的双重含义：
- **弦直而有力** = 紧脉（寒束肌表，或冲气上逆）
- **弦直而无起伏、不滑润** = 假有力（脾胃真气外泄，或肝木横恣）

### 2.3 洪滑之象

> **"洪者如波涛叠涌，势作起伏；滑者指下滑润，累累如贯珠。"**

洪滑 = 真有力的两个标志：
- **洪**：搏动有起伏波动，象征气机通畅
- **滑**：指下有圆润流畅感，象征津液充足

**无洪滑之象而有弦硬者** = 阴虚火实之脉（而非阳明实热），用白虎汤时必须加人参。

### 2.4 弦硬：假有力的最高级形态

弦硬是张锡纯出现频率最高的复合脉象，辨证含义远较一般弦脉丰富：

| 弦硬的具体形态 | 辨证 | 治则 |
|----------------|------|------|
| 大而弦，按之似有力，非真有力 | 脾胃真气外泄 + 冲脉逆气上干 | 镇摄汤 |
| 弦硬而长，右部尤甚 | 冲气上冲 + 胃气上逆 | 赭石为主降逆 |
| 左关弦硬 | 肝胆之火炽盛 | 清肝 |
| 左右皆弦硬 | 肝火+冲胃气逆（脑充血之兆） | 建瓴汤/镇肝熄风汤 |
| 弦硬而大 | 真气不敛、内风煽动 | 龙骨牡蛎山萸肉 |
| 弦硬有力，重按不实 | 血亏而气亦亏 | 滋阴+补气 |
| 弦硬且数 | 阴分亏损（弦硬=阴亏之象） | 大滋真阴 |

## 三、大气下陷脉法：张锡纯脉诊的轴心

大气下陷是张锡纯辨证体系中最大的原创贡献，其脉法自成一体。

### 3.1 大气下陷的定义与脉象总则

> **"胸中大气亦名宗气……《内经》谓宗气积于胸中以贯心脉而行呼吸，深思《内经》之言，知肺叶之辟阖，固为大气所司，而心机之跳动，亦为大气所司也。"**

大气即宗气，主管：
1. 肺之呼吸（肺叶辟阖）
2. 心之跳动（心机跳动）
3. 脉之起伏（贯心脉）

当大气下陷时，三者皆受影响，脉即失去正常形态。

### 3.2 大气下陷的核心脉象

> **"其脉象沉迟微弱，关前尤甚。其剧者，或六脉不全，或参伍不调。"**

四层脉象特征：

1. **关前微弱**：是最核心的定位性脉象——"脉关前微弱不起"，关前（寸部）对应胸中大气所在。
2. **沉迟**：大气下陷之本脉，因大气无力鼓动血脉——"大气下陷之脉大抵皆迟"。
3. **微弱/沉濡**：气陷不能鼓脉——"大气下陷，不能鼓脉外出，以成起伏之势"。
4. **指下不觉其动**：气陷至极——"其脉非微细，而指下若不觉其动"。

### 3.3 大气下陷与其他证候的脉象鉴别

| 证候 | 脉象 | 关键鉴别点 |
|------|------|-----------|
| 大气下陷 | 沉迟微弱，关前尤甚 | 肩下垂，不肩息 |
| 阴虚不纳气作喘 | 多数，尺弱寸强 | 肩息（耸肩） |
| 寒饮结胸 | 脉似寒凉，且畏寒凉 | 畏惧寒凉 |
| 大气下陷（假寒象） | 脉似寒凉，但不畏寒凉 | 不畏寒凉，只觉短气 |

> **"不纳气作喘者，其脉多数，或尺弱寸强；大气下陷之喘，其脉多迟而无力，尺脉或略胜于寸脉。"**

### 3.4 兼证对脉象的影响

> **"其脉多迟，而又间有数者，或因阴分虚损，或兼外感之热，或为热药所伤，乃兼证之现脉，非大气下陷之本脉也。"**

这解释了为什么大气下陷患者的脉有时反而数——那是兼证（阴虚、外热）叠加所致，本质仍是气陷。

### 3.5 左右脉差异的机制

> **"肺之脉诊在右部，故大气下陷，右部之脉多微弱者其常也。然人之元气自肾达肝，自肝达于胸中，为大气之根本。其人或肝肾素虚，或服破肝气之药太过，其左脉或即更形微弱。"**

- 右脉微弱 = 大气下陷最常见
- 左脉微弱 = 肾肝元气（大气之根）受损
- 左右皆微弱 = 根与本皆损

### 3.6 虚里动脉：大气搏动的体表窗口

张锡纯继承《内经》"胃之大络，名曰虚里，贯膈络肺，出于左乳下，其动应衣，脉宗气也"，将此发展为诊大气盛衰的重要体征：

> **"虚里之络，即胃输水谷之气于胸中，以养大气之道路。而其贯膈络肺之余，又出于左乳下为动脉。是此动脉，当为大气之余波。"**

虚里（心尖搏动处）与脉诊的关系：
- 健康人虚里搏动微弱不可见——大气盛而冲气不逆
- 冲气上冲时——"循虚里之大络贯膈络肺，复出于左乳下与动脉相并，以致动脉因之大动"
- 革脉危证时——"周身脉管皆大动，几有破裂之势"（虚里搏动 + 六脉弦硬同步增强）

诊断意义：虚里大动 = 冲气挟大气上逆，属肝木横恣的体表信号，与左脉弦长同步出现。

## 四、三部九候的脏腑定位

### 4.1 关前/关后对比

张锡纯大量使用关前/关后对比来做精确辨证：

| 脉象 | 辨证 | 方剂 |
|------|------|------|
| 关前微弱不起 | 胸中大气下陷 | 升陷汤 |
| 关前微弱 + 关后似有力 | 大气下陷兼伏邪化热 | 升陷汤+石膏 |
| 寸盛尺虚 | 血随气升，脑充血之兆 | 建瓴汤 |
| 尺部无根 | 元气将脱 | 急补敛（山萸肉） |
| 尺部重按不实 | 下焦气化不固 | 加萸肉/龙骨/牡蛎 |

### 4.2 左右脉辨证：核心框架

张锡纯的左右辨证极为精细且贯穿全书：

| 脉位 | 对应 | 常见脉象 → 辨证 |
|------|------|-----------------|
| **左脉** | 肝胆（肝气化先行于左） | 弦硬 = 肝火/肝风；弦长 = 肝盛；弦细 = 肝血虚 |
| **右脉** | 肺胃 + 冲气 | 弦硬而长 = 冲气挟胃气上逆；微弱 = 大气下陷；洪实 = 阳明腑热 |
| **左弦（弦硬）+ 右洪（洪实）** | 肝火 + 阳明热 | 白虎加人参汤 |
| **左弦硬 + 右弦长** | 肝火 + 冲胃气逆 | 镇肝降胃 |

> **"其左脉弦硬者，阴分亏损而肝胆有热也；右部弦长者，因冲气上冲并致胃气上逆也。"**
> ——这是张锡纯最常见的左右辨证模板

### 4.3 冲气上冲的脉象特征

> **"方书论脉有直上直下冲脉昭昭之语，所谓直上直下者，即脉弦且长之形状也。"**

右脉弦长 = 冲气上冲的标志性脉象。弦长而有力者兼有外感热入阳明。

## 五、特殊脉象域

### 5.1 散乱脉 = 元气将脱

> **"脉散乱如水上浮麻，不分至数，此将脱之候也。"**
> **"六脉散乱无根，有将脱之象。"**

- 散乱：脉象失去规律，如浮麻漂浮水面
- 无根：尺部按之即无
- 治疗：急用山萸肉收敛固脱

### 5.2 紧脉 = 弦而有力

> **"脉无起伏为弦，弦而有力，即紧脉也。"**

紧脉实为弦的一种——无起伏且有力。多见寒束肌表，或大气下陷时内有实热不能透达。

> **"此证因大气下陷，虽内有实热，不能鼓脉作起伏之势，故不为洪而为紧，且为沉紧也。"**

### 5.3 结代/雀啄脉

> **"结脉现象未必皆属内亏，恒有因气分不舒，理其气即可愈者。又有脉非结代，而若现雀啄之象者，此亦气分有所阻隔也。"**

非死证——气分郁滞可致脉律不整，理气可愈。

### 5.4 弦而无力

| 脉象 | 辨证 |
|------|------|
| 弦而无力 | 病久而气化虚 |
| 弦大而兼芤 | 失血过多 |
| 左脉细弱 + 右脉弦而有力 | 病久生热，胃气因热上逆 |

### 5.5 数脉的气虚维度

> **"脉数者，固系阴虚，亦系气分虚弱，有不能支持之象。"**

> **"元气虚极莫支者，其脉可至极数。设有人或力作，或奔驰，至气力不能支持之时，其脉必数。"**

### 5.6 革脉：弦硬假有力的终极形态

革脉是张锡纯在医论中专篇论述的唯一脉象（第57篇《论革脉之形状及治法》），定义为弦硬发展到危险边缘的形态：

> **"革脉最为病脉中之险脉……其形状如按鼓革，外虽硬而中空，即弦脉之大而有力者。因其脉与弦脉相近，是以其脉虽大而不洪（无起伏故不洪），虽有力而不滑（中空故不滑）。"**

革脉 = 弦硬假有力的最高级形式：
- 外观：大而硬，按之如按鼓皮——外绷紧内空虚
- 与弦硬的关系：弦硬→脉管坚实、有韧性；革脉→脉管已失弹性，外强中干
- 核心特征：大而不洪 + 有力而不滑（二者皆无 = 胃气已竭）
- 病机：**阴阳离绝，上下不相维系**
- 命名：脉至如此，病将变革，危在顷刻

**治案逻辑**（三次转折）：
1. 初诊：脉大而硬、微兼洪象 → 白虎加人参汤（清外感热，洪象消但大硬如故）
2. 大便通下后陡变：脉之大与硬较前加两倍，周身脉管皆大动 → 改用萸肉、龙骨、熟地、山药、野台参、白术、炙甘草（大剂收敛固脱）
3. 三剂后脉象近平和，续调而愈

核心教训：革脉不能用攻下猛剂，虽见便结腹满也不可峻下，下后假有力崩解、真气将脱。革脉出现时，"收敛固脱"优先于"通下攻邪"。

### 5.7 促脉：改步武而不止

张锡纯对促脉有独到的非传统解释，反驳"脉动速时一止"的旧说：

> **"促脉与结、代之脉皆不同，注疏诸家多谓脉动速时一止者曰促。夫促脉虽多见于速脉之中，而实非止也。譬如人之行路，行行且止，少停一步复行，是结、代也。又譬如人之奔驰，急急速走，路中偶遇不平，足下恒因有所龃龉，改其步武，而仍然奔驰不止，此促脉也。"**

促脉 ≠ 有歇止。促脉本质是：
- 脉率极速（奔驰） + 脉律不规则（改步武） + 不停歇（仍然奔驰不止）
- 结/代：行行且止，真停顿
- 促：改步武而不止，节奏紊乱但不中断

> **"是以促脉多见于速脉中也。凡此等脉，多因外感之热内陷，促其脉之跳动加速，致脉管有所拥挤，偶现此象，名之为促，若有人催促之使然也。"**

病因：外感热邪内陷 → 脉管拥挤 → 跳动加速且节律不齐。治当清热透表，热透出则促自消。方中重用芩、连清内陷之热（如葛根黄芩黄连汤证）。

### 5.8 浮脉的关前关后分部诊断

张锡纯解释桂枝汤"阳浮而阴弱"时，将浮脉与关位结合做精确分部辨证：

> **"脉法关前为阳，关后为阴，其浮脉见于关前，弱脉见于关后。浮者着手即得，弱者不任重按。"**

- 关前（寸）浮 = 阳浮，风邪在表，卫气抗争
- 关后（尺）弱 = 阴弱，营气不足，不任重按

对桂枝汤证脉象的独特解读：
> **"其未病之先，不但关后之脉弱，即关前之脉亦弱。既病之后，其关前脉之弱者转为浮脉所掩，而不见其弱耳。然其脉虽浮，必不任重按，是浮中仍有弱也。"**

——浮只是表层假象，重按之下弱脉仍在。故桂枝汤中需加黄芪补胸中大气，加薄荷助出汗。

### 5.9 弦无力：病久气化虚

弦无力是弦硬的对立面，代表久病气化已衰：

| 脉象 | 兼证 | 辨证 |
|------|------|------|
| 脉象和平，重按微弦无力 | 胁疼 + 胸满 + 咳嗽 | 初疑气郁，投理气药反加重——实为大气下陷，因感冒后力田伤气 |

## 七、张锡纯脉诊思维总纲

### 7.1 四个判断层次

张锡纯看一个脉象时，按以下层次递进判断：

**第一层：起伏？**
- 有起伏 → 洪象，气机通畅
- 无起伏 → 弦象，气机郁滞或大气虚陷

**第二层：滑润？**
- 有滑润 → 津液充足
- 无滑润 → 津枯或气滞

**第三层：数迟？**
- 数 → 先问气分强弱（非纯阴虚）
- 迟 → 先问大气是否下陷（非纯寒凉）

**第四层：有力之真假？**
- 敦浓和缓中见有力 → 真有力，脾胃壮旺
- 弦直无起伏中见有力 → 假有力，肝木横恣或大气外泄

### 7.2 张锡纯的脉诊口诀（从原著提炼）

1. **脉无起伏 = 弦**；弦而有力 = 紧；弦而无力 = 虚寒
2. **洪滑兼具** = 真有力（有胃气）
3. **弦直无洪滑** = 假有力（肝横或气泄）
4. **关前微弱** = 大气下陷的第一线索
5. **右脉弦长** = 冲气上冲（右属胃，弦长属冲）
6. **左脉弦硬** = 肝火/肝风/阴亏（左属肝）
7. **脉数至七八至不绝望**——问气分
8. **脉迟未必寒**——先排除大气下陷
9. **六部皆有和缓 = 正象**（脾土之气濡养四脏）
10. **尺部无根 = 脱候**（无论脉象其他特征如何）
11. **大而不洪 + 有力不滑 = 革脉**（阴阳离绝之险，禁峻下，急收敛）
12. **促 ≠ 止**——改步武而不止，外热内陷致脉管拥挤，当清透
13. **浮脉关前见**——"阳浮"在表；关后弱——"阴弱"在里
14. **虚里大动** = 冲气挟大气上逆，与左脉弦长同步出现

### 7.3 巍哥，如你让张锡纯把脉，他会这样思考：

1. 先摸：脉有没有像波涛一样的起伏？（洪象）
2. 再看：指下圆润流畅吗？（滑象）
3. 如果没有起伏——这是弦脉，已非正象
4. 如果有起伏且有滑润——这是真有力，可放心
5. 关前部位是否比关后弱？（大气下陷的定位信号）
6. 左脉和右脉有没有明显差异？（左肝胆 vs 右肺胃+冲气）
7. 若是数脉——别急着清火，先看气分有没有撑不住的迹象
8. 若是弦硬而长——注意冲气上冲（右）或肝风内动（左）
9. 尺部按下去能感觉到吗？按之即无 → 速用山萸肉固脱
10. 整体有没有一丝和缓之意？全无和缓 → 脾胃真气已衰
11. 若大而硬却无洪滑 → 这是革脉，阴阳离绝之兆，先收敛固脱，不可攻下
12. 若脉急速但不规则却不停止 → 促脉，外热内陷，清热透表为主
13. 摸一下左乳下虚里——搏动是否异常放大？若跳动剧烈 + 左脉弦长 = 冲气上冲
