# 陈建国《仲景阴阳脉法》体系

> 陈建国，胡希恕名家研究室主任，师从冯世纶（胡希恕传人）。著有三部书：
> - 《仲景阴阳脉法》（2020）
> - 《经方脉证图解》（2021，ISBN 9787513269926）
> - 《神农升降药法》（2021）
>
> 本文档整合巍哥口述原著原文及网络公开资料，独立存储陈建国脉证体系全部内容。

## 一、核心理论

| 项目 | 内容 |
|------|------|
| 总纲 | "夫阳盛阴虚，汗之则死，下之则愈。"（陈称仲景用了全书4个感叹句中3个来强调此大法） |
| 左手候阴 | 阴津血之盛衰。阴盛→升法。阴虚→甘寒补阴 |
| 右手候阳 | 阳气之盛衰。阳盛→降法。阳虚→甘温补阳 |
| 口诀 | 实则左升右降，虚则右升左降 |

## 二、操作步骤

1. **先定总体太过/不及**：太过=脉实证实；不及=脉虚证虚
2. **找最强脉位**：左脉盛→阴盛→升法；右脉盛→阳盛→降法
3. **相对太过脉**：总体不及中有一部相对突出→治疗方向与总体太过相反
4. **诊断指标**："独异者病"——以脉动力量+脉管粗细判断，于最异常处定位病机

## 四、附录：各类方脉证鉴别（巍哥口述原著原文）

### 麻黄类方脉证鉴别

陈建国原著就麻黄类方的脉证鉴别给出了详细比对。以下四方不在50经方目录中，但从鉴别角度与麻黄汤有直接临床对比价值。

**鉴别维度**：是否浮紧、浮紧程度及范围、脉动的力量、是否兼夹太过脉。

**麻黄汤**（已是核实方）
- 左手寸脉太过脉浮紧。临床双侧均可出现，但对比一定是左侧较右侧更有力更显著。

**麻黄连轺赤小豆汤**
- 左手寸脉出现太过脉；但右侧寸部同样有太过脉。
- 右寸部太过脉表现为数急有力而非拘急，重按右寸太过脉应指仍旧有力。

**麻黄杏仁甘草石膏汤**
- 左寸浮缓，应指并不拘急。
- 右手脉更为有力（与麻黄汤鉴别关键）。
- 兼以症状表现的区别很容易鉴别。

**大青龙汤（两个方向）**
- 方向一（发散表寒）：左寸浮紧的同时，左侧关部甚至尺部也会有一定程度的浮紧，寸部最为突出。比麻黄汤更偏于"阴阳俱紧"，浮紧脉也更加的数急躁动。右手脉更为有力。
- 方向二（治疗溢饮）：可表现为"浮缓"——"浮而濡"，浮而边界不清。兼身重、身肿。右手脉更为有力。

**麻黄加术汤**
- 治疗溢饮时，浮位脉象更拘急者用麻黄加术汤。
- 鉴别：浮位脉管边界不清者用大青龙汤（方向二）。

**小青龙汤**（已是核实方）
- 咳喘时，左侧关部兼有水饮脉。

> 说明：以上鉴别内容为巍哥口述陈建国原著原文，已核实。麻黄连轺赤小豆汤、麻杏甘石汤、大青龙汤、麻黄加术汤四方不在《经方脉证图解》50方目录中（陈建国或另有论述，待核）。

### 小柴胡汤类方脉证鉴别

**小柴胡汤**（已是核实方）
- 左关脉中位太过（弦），往往左手寸、尺部反而无力，寸尺部力量汇聚到关部。左关部太过脉范围局限（较短），类似应指有力的"绿豆"。中位应指向上弹手明显为特征。

**柴胡加龙骨牡蛎汤**（不在50方目录中）
- 除了具有小柴胡汤证左手脉特征以外，右关部、寸部出现应指有力的太过脉。

**四逆散**（在50方目录第二章第四节，已核实升级完整版）
- 左关细弦——即左手脉较小柴胡汤脉证更细些。左关中位弦而有力（与小柴胡汤脉证同），气滞为主血虚为次。后世逍遥散脉证基本相似。

**大柴胡汤**（在50方目录第二章第二节，已核实）
- 左关太过脉（中位，弦）伴右关太过脉（滑、数），"双关弦"。与小柴胡汤核心鉴别：大柴胡汤双手关部均太过（左关弦＋右关滑数），小柴胡汤仅左关太过。左关太过程度＞右关太多，体现阴盛为主兼阳盛的病机。
- 《伤寒论》无大黄版右关太过脉力量更小；右手太过脉也可出现在尺部或关尺部。

**柴胡桂枝干姜汤**（不在50方目录中）
- 除了具有小柴胡汤的左手脉特征以外，伴有右手尺部显著无力的不及脉。

> 说明：柴胡加龙骨牡蛎汤、柴胡桂枝干姜汤不在《经方脉证图解》50方目录中。

### 麻黄细辛附子汤脉证鉴别

**麻黄细辛附子汤**（已是核实方）
- 左手尺部的沉位太过脉，余脉均沉而无力。尺部沉紧或沉弦。仲景书言少阴证脉"微细"乃指整体，进一步探查左手尺部沉位则明确候得有力的太过脉。

**四逆汤**（已是核实方）
- 诸脉沉而无力，以右手尺部最为无力为特征。与麻黄细辛附子汤核心鉴别：麻附辛汤左尺沉位有力（实寒为主），四逆汤右尺最无力（虚寒为主）。

**麻黄附子甘草汤**（不在50方目录中）
- 无论病机、方证、脉证均与麻黄细辛附子汤非常接近。两方区别点：从右侧尺部沉位的脉动力量体会，麻黄附子甘草汤相对更为无力。更虚者加甘草，下焦邪气明显者加透发的细辛，脉证上客观反映了两方证病机的不同。

**大承气汤**（已是核实方）
- 临床仅从脉证而论，当与大承气汤鉴别，详见大承气汤条。

### 瓜蒂散脉证鉴别

**瓜蒂散**（已是核实方）
- 左手寸部浮位、中位均出现有力的太过脉（滑、数、紧）。关键鉴别：探及浮位太过脉后继续寻至中位，仍可触及非常有力的脉动——此与桂枝汤、葛根汤、麻黄汤以浮位最强、由浮至沉依次显著减弱截然不同。
- 与吴茱萸汤鉴别：两者相比，吴茱萸汤的左手关部脉动更为有力。临床中结合方证有利于此方应用进行鉴别。
