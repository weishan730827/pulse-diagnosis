# 各家辨证勾选表设计详情

## 1. 胡希恕勾选表（已完成 v1）

**三步递进锁定**：
Step1 八纲定位 → Step2 六经内症候群聚类 → Step3 脉象加权确认
每个分型标签映射到对应方证模板，如"恶寒发热+无汗+身疼+脉浮紧"→麻黄汤证

**独有特点**：按临床分型组织（伤寒型/中风型/蓄水型/结胸型…），勾选一个类型=锁定一组方证

---

## 2. 郑钦安勾选表

**辨证逻辑**：辨阴证阳证 → 直取病机要害
**核心三问**（勾选项即结论方向）：

| 维度 | 阳证 | 阴证 |
|------|------|------|
| 渴饮 | 渴喜冷饮 | 口不渴/喜热饮 |
| 舌象 | 舌干/舌红 | 舌润/舌滑/舌淡 |
| 脉象 | 脉浮/数/大/滑/实有力 | 脉沉/微/细/弱无力 |

**扩展项**：
| 维度 | 勾选项 |
|------|--------|
| 二便 | 小便清长 / 小便短赤 / 大便溏 / 便秘 |
| 神志 | 但欲寐 / 烦躁 / 精神如常 |
| 寒热 | 恶寒发热 / 但热不寒 / 但寒不热 / 午后发热 |
| 汗 | 自汗 / 盗汗 / 无汗 / 大汗 |
| 头面 | 面赤 / 面白 / 头痛 |

**设计特点**：输入最少（7-8个核心勾选），输出阴阳判断+扶阳/滋阴方向

---

## 3. 张仲景勾选表

**辨证逻辑**：六经提纲症 + 方证对应
**核心设计**：六经各一组提纲勾选

| 六经 | 提纲勾选项 |
|------|-----------|
| 太阳 | 脉浮 / 头项强痛 / 恶寒 |
| 阳明 | 身热汗自出 / 不恶寒反恶热 |
| 少阳 | 口苦 / 咽干 / 目眩 / 往来寒热 / 胸胁苦满 |
| 太阴 | 腹满而吐 / 食不下 / 自利益甚 / 时腹自痛 |
| 少阴 | 脉微细 / 但欲寐 |
| 厥阴 | 消渴 / 气上撞心 / 心中疼热 / 饥不欲食 |

**扩展**：按六经细化症状（与胡希恕step2共用数据），再加方证直接对应

---

## 4. 姚梅龄勾选表

**辨证逻辑**：纯脉诊为核心，症状为辅
**设计特点**：
- 知医邦28脉量化表是姚梅龄体系的主输入
- 脉位（寸关尺两部/三部）+ 脉象形态（28类）= 姚梅龄诊断引擎的全部输入
- 可勾选补充症状以辅助方证匹配
- **本质上是知医邦脉诊表的直接消费者**

---

## 5. 张锡纯勾选表

**辨证逻辑**：辨气机升降 → 对应升陷/镇逆类方
**核心二选一**：

| 方向 | 勾选项 |
|------|--------|
| 大气下陷 | 气短不足以息 / 胸闷 / 心中怔忡 / 脉沉迟微弱 |
| 气机上逆 | 呃逆 / 呕吐 / 咳喘 / 吐血衄血 / 脉弦长有力 |

**扩展项**：乏力 / 懒言 / 自汗 / 怔忡 / 头晕目眩

---

## 6. 刘渡舟勾选表

**辨证逻辑**：按"十论"分诊——哪个维度有异常就展开哪个维度
**分论勾选入口**：

| 论 | 关键勾选项 |
|----|-----------|
| 水证 | 小便不利 / 渴欲饮水 / 水入即吐 / 头眩 |
| 火证 | 心烦 / 口苦 / 舌红 / 脉数 |
| 气证 | 腹胀 / 心下痞 / 噫气 / 气上冲 |
| 血证 | 少腹急结 / 经血异常 / 瘀斑 |
| 痰证 | 咳痰 / 胸闷 / 苔腻 |
| 食证 | 食不下 / 嗳腐吞酸 |

---

## 7. 曹颖甫勾选表

**辨证逻辑**：症状→经方直查（方证对应，原方72%不加减）
**设计特点**：
- 最简洁——勾选项就是经方的直接触发器
- 如"项背强几几"→葛根汤，"心下痞"→泻心汤类
- 类似于方证数据库的倒排索引前端

| 症状组 | 直接对应方 |
|--------|-----------|
| 项背强几几 | 葛根汤 |
| 胸胁苦满 | 小柴胡汤 |
| 心下痞 | 泻心汤类 |
| 腹痛 | 桂枝加芍药汤 |
| 咳喘 | 小青龙汤 / 麻黄杏仁甘草石膏汤 |

---

## 8. 黄元御勾选表

**辨证逻辑**：一气周流，中气为枢 → 四维（肝心肺肾）病变皆因中气不运
**核心轴（必勾）**：

| 维度 | 勾选项 |
|------|--------|
| 中气（脾） | 纳呆 / 腹胀 / 大便溏 / 完谷不化 |
| 脾胃升降 | 呕吐 / 呃逆 / 下利 |
| 四维波及 | 胁痛（肝）/ 心悸（心）/ 咳喘（肺）/ 腰酸（肾）|

**扩展**：舌象（知医邦共用）+ 畏寒 / 发热 + 脉象（知医邦共用）

---

## 9. 陈建国勾选表

**辨证逻辑**：三部九候阴阳脉法——50方脉位签名对照
**核心设计**：寸关尺 × 浮中沉 3×3 网格

|   | 寸 | 关 | 尺 |
|---|----|----|-----|
| 浮 | □ | □ | □ |
| 中 | □ | □ | □ |
| 沉 | □ | □ | □ |

每个格填入脉象（如"弦""弱""滑"），形成9格脉象签名 → 与50方签名库匹配

**设计特点**：完全独立于其他体系，唯一输入是9格脉象签名 + 可选症状辅助
