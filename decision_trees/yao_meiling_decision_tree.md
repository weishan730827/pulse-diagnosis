## 核心方法

### 独处藏奸

> "六部脉象中某部独大、独小、独弦、独弱、独滑、独涩——是辨证关键。"
> 不是所有脉象的平均，而是找那个"不合群"的脉。

| 步骤 | 操作 |
|------|------|
| 1 | 六部全面候脉（左右×寸关尺），浮中沉三部 |
| 2 | 标记每部异常维度及其程度（+~+++++） |
| 3 | 突出"某部独×"的异常（如：六部中唯左关独弦） |
| 4 | 识别上盛下虚/下盛上虚等三部间关系 |
| 5 | 若多部同一种异常，判别程度差异 |

### 脉症合参

姚梅龄不主张"脉症从舍"——所谓假脉假症是从另一侧面反映疾病本质。脉症不符时，某些脉症提示更深层次的发病机理。

### 以脉象形成机理代替脉象主病

传统脉象主病过于机械（如"浮主表"）。姚梅龄按临床出现概率排序，从常到变：
- 浮脉最常见=表证（概率最高）
- 浮脉也可能=虚阳外浮、里热蒸腾（概率较低）
- 不排除但标注概率

## 脉象分类体系（七大类别·37脉）

| 类别 | 脉象 | 姚氏解读要点 |
|------|------|-------------|
| 脉率异常 | 数、疾、迟、缓 | 数脉>79次/分（原文确认） |
| 脉律异常 | 代、结、促、乍数乍疏、三五不调 | 促=数中一止（非胡希恕定义） |
| 脉位异常 | 浮、沉、伏 | 浮≠必表证，需结合力度 |
| 脉体异常 | 细、大、短、长、曲 | 细=如小提琴第二弦，大>15mm |
| 脉力异常 | 实、弹指、虚、弱、微、无脉、无根 | 弹指脉=危重信号 |
| 脉势异常 | 滑、涩、弦、紧、硬、软 | 弦≠少阳，需结合部位 |
| 复合脉 | 浮大中空、革、牢、洪、动、芤、濡、散、不静、静、上盛下虚、下盛上虚 | 上盛下虚=高血压类；下盛上虚=下肢水肿类 |

## 六经皆有表证（姚梅龄贡献）

| 六经 | 表证脉象 | 说明 |
|------|----------|------|
| 太阳之表 | 浮脉 | 浮取太过，邪犯太阳 |
| 少阴之表 | 沉脉（沉而细或沉而微） | 邪闭少阴之表 |
| 厥阴之表 | 沉脉/弦脉 | 邪闭厥阴+风木之象 |
| 阳明之表 | 浮数/浮洪或沉实 | 表里俱热则沉 |
| 少阳之表 | 弦细 | 半表半里之浅层 |
| 太阴之表 | 浮缓/浮弱或沉缓 | 太阴为里，其表在肌肉四肢 |

## 重点脉象诊断意义

| 脉象 | 诊断要点 |
|------|----------|
| 涩脉 | 血瘀（最常见）/精亏/气滞 |
| 弦脉 | 气滞（最常见）/水饮/少阳 |
| 浮脉 | 表证（最常见）/虚阳外浮 |
| 沉脉 | 里证（最常见）/少阴之表 |
| 弹指脉 | 肝阳上亢/阳气暴脱（危重） |
| 无根脉 | 肾气衰败（危重） |
| 上盛下虚 | 寸关有力+尺部无力——肝阳上亢/高血压 |
| 下盛上虚 | 尺部有力+寸关无力——水湿下注 |

## 特殊病脉转变诊断

| 脉象变化 | 临床意义 |
|----------|----------|
| 脉骤停 | 正气将脱（危） |
| 脉出（无→有） | 阳气来复（好转） |
| 脉沉见起 | 邪气出表（好转） |
| 脉转大 | 邪气膨胀（加重） |
| 弦转软 | 肝气舒畅（好转） |
| 脉紧实转微 | 正气已衰（危） |
| 脉转不柔和 | 胃气将绝（危） |
