# 张锡纯辨证方法论——原著提炼的完整决策树

> 源于《医学衷中参西录》原著，非网络解读。张锡纯辨证的根本方法是：以"阴常不足"为病机观，以气机升降为框架，以三部总看脉法定虚实，以左右对比定脏腑，最终在自创方和仲景方两大方池中精准锁定。

---

## 一、根本病机观：阴常不足，阳常有余

**原著出处**：《驳方书贵阳抑阴论》

> "若论已病，又恒阳常有余，阴常不足……医者当调其阴阳，使之归于和平，或滋阴以化阳，或泻阳以保阴，其宜如此治者，又恒居十之八九。"

阳常有余非真阳盛——乃**阴虚不能敛阳，阳浮于外**。

治则铁律：
- 滋阴以化阳，泻阳以保阴
- 即使阳虚用桂附，亦必少佐滋阴——"上焦阴分不虚而后可受参芪，下焦阴分不虚而后可受桂附"
- 外感病中伤寒中于阴经宜热药者"百中无二三"
- 内伤病中虚劳、吐衄、淋痢、黄疸、肿胀、喘嗽，"阳盛阴虚者实为十之八九"

## 二、辨证五步决策树

```
病人就诊
  │
  ├─【第一步：问主症，定气机升降大方向】
  │     │
  │     ├─ 气短/呼气困难/乏力/下坠感 →【下陷路径】
  │     │     │
  │     │     └─【脉诊确认】
  │     │           ├─ 右寸关沉迟微弱 → 大气下陷
  │     │           │     ├─ 兼心肺阳虚（身冷背紧恶寒）→ 回阳升陷汤
  │     │           │     ├─ 兼气分郁结（胸中满痛胁胀）→ 理郁升陷汤
  │     │           │     ├─ 兼脾气虚极（小便不禁）→ 醒脾升陷汤
  │     │           │     └─ 单纯大气下陷 → 升陷汤
  │     │           │
  │     │           └─ 脉沉细无力但非寸关特甚 → 中气下陷（非大气下陷）
  │     │                 → 补中益气汤方向
  │     │
  │     └─ 气上冲/呃逆/呕吐/头胀/胸满 →【上逆路径】
  │           │
  │           └─【脉诊确认】
  │                 ├─ 右脉弦硬而长（大于左脉）→ 冲气上冲
  │                 │     ├─ 兼肝火盛（左脉弦硬）→ 镇肝熄风汤
  │                 │     ├─ 兼胃热（右脉洪滑）→ 白虎汤+赭石
  │                 │     ├─ 阴分虚甚 → 参赭镇气汤
  │                 │     ├─ 寒热夹杂 → 寒降汤/温降汤
  │                 │     └─ 冲胃并逆 → 镇逆汤
  │                 │
  │                 └─ 脉非弦长、关尺沉 → 需进一步辨六经
  │
  ├─【第二步：脉诊定虚实寒热】
  │     │
  │     ├─ 三部总按：有力？无力？
  │     │     ├─ 无力 → 虚证，补益为主
  │     │     └─ 有力 → 辨真假
  │     │           ├─ 真有力（洪滑起伏，如波涛叠涌/累累如贯珠）→ 实热实证 → 清泻
  │     │           └─ 假有力（弦直无起伏，虽大而有力）→ 阴虚阳浮 → 滋阴潜阳
  │     │
  │     ├─ 浮沉：表里上下
  │     │     ├─ 浮 → 表证或阴虚阳浮（非仅外感专属）
  │     │     └─ 沉 → 里证或大气下陷
  │     │
  │     └─ 数迟：寒热（不泥古）
  │           ├─ 数而有力 → 真热
  │           ├─ 数而无力 → 虚（大补元气，忌寒凉）
  │           ├─ 迟而有力 → 寒积
  │           └─ 迟而无力 → 大气下陷
  │
  ├─【第三步：左右对比定脏腑偏颇】
  │     ├─ 右脉 > 左脉 → 冲气上冲、胃气不降 → 镇逆降胃
  │     ├─ 左脉 > 右脉 → 肝阳上亢、阴虚火旺 → 平肝滋阴
  │     ├─ 左脉沉细 → 肝气郁结不升 → 升肝舒郁
  │     └─ 右寸关沉 → 胸中大气下陷 → 升陷补气
  │
  └─【第四步：尺部定根，决预后】
        ├─ 尺脉有根（重按有力）→ 可治
        └─ 尺脉无根（重按即无）→ 危证，急固脱
```

## 三、脉诊三铁律

| 铁律 | 内容 | 原著依据 |
|------|------|----------|
| **有力无力定虚实** | 三部总按，有力→实证/热证，无力→虚证/气陷 | "有力无力，虚实立判；虚实一明，治则无差" |
| **辨真假有力** | 真有力有洪滑之象（波涛叠涌、累累如贯珠），弦直无起伏为假实（阴虚阳浮），误攻则"凶危立见" | "若脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力" |
| **数脉主虚不主热** | 数而无力→大补元气，忌寒凉。数而有力方为真热 | "见数脉便用凉药，杀人无数" |

## 四、脉象-病机-方证完整对应

### 4.1 弦脉——核心脉象，主肝肾阴虚

弦是张锡纯最重视的脉象，集中体现"阴常不足"的根本病机。

| 弦脉细分 | 病机 | 临床表现 | 治法方向 |
|-----------|------|----------|----------|
| 左脉弦细无力 | 肝血虚、大气下陷 | 气短、乏力、产后下血 | 升陷方向 |
| 左脉弦硬有力长 | 肝肾阴亏、阴虚不能潜阳、冲气上冲 | 吐血、咳逆、中风 | 镇逆+滋阴 |
| 右脉弦细无力 | 土为木伤、脾胃失运 | 胁痛、胃脘痛 | 健脾+疏肝 |
| 右脉弦硬有力长（>左脉） | 冲气上冲、胃气不降 | 呃逆、呕吐、膈食 | 镇冲降胃 |
| 左右脉弦细无力 | 气血两亏、阴阳两虚 | 痢疾、肠炎 | 双补气血 |
| 左右脉弦硬有力长 | 阴分有亏、肝木过盛 | 久痢阴虚、膈食 | 滋阴平肝 |

**原著原文**：

> "弦脉主肝血少、肝阴虚、肝火浮动。左脉弦，已阴虚不能潜阳，肝火浮动；右脉弦，已肝火与肝气并冲、胃上逆。兼数者热甚，兼沉者肝气郁。细而无力者气虚，硬而有力者虚火盛。"

### 4.2 完整九脉辨证表

| 脉象 | 部位 | 病机 | 对应方剂方向 |
|------|------|------|-------------|
| 弦细无力 | 左 | 肝血虚、大气下陷 | 升陷汤类 |
| 弦硬有力长 | 左 | 肝肾阴亏、冲气上冲 | 镇逆+滋阴 |
| 弦硬有力长 | 右（>左） | 冲气上冲、胃气不降 | 参赭镇气汤/镇逆汤 |
| 弦细无力 | 右 | 土为木伤 | 健脾疏肝 |
| 浮弦有力（按不实） | 左 | 肝血虚损、肝火上升 | 滋阴潜阳 |
| 浮芤 | 右 | 气血亏极、阳气上浮 | 补气血 |
| 浮而无力 | 左右 | 失血过多、下焦不固 | 补血固涩 |
| 沉迟微弱 | 右寸关 | 大气下陷 | 升陷汤 |
| 沉细 | 左右 | 心肺阳虚 | 温阳 |
| 数而无力 | 左右 | 阴虚、气虚 | 大补元气 |
| 数而有力 | 左右 | 真热 | 清泻 |
| 数而沉濡 | 左右 | 阴虚不能纳气 | 滋阴纳气 |
| 洪滑有力 | 右 | 阳明实热、胃气不降 | 白虎汤清降 |
| 滑而有力 | 左右 | 湿热壅滞 | 清热化痰 |
| 尺脉无根 | 尺部 | 肝肾虚极、阴阳不维 | 急固脱 |

## 五、大气下陷 vs 气逆之喘——原著关键鉴别

这是张锡纯独创的最重要临床鉴别，选错方向则"凶危立见"。

**原著原文**：

> "凡喘证，无论内伤外感，其剧者必然肩息；大气下陷者，虽至呼吸有声，必不肩息。盖肩息者，因喘者之吸气难；不肩息者，因大气下陷者之呼气难也。欲辨此证，可作呼气难与吸气难之状，以默自体验，临证自无差谬。又喘者之脉多数，或有浮滑之象，或尺弱寸强；大气下陷之脉，皆与此成反比例。"

| 鉴别点 | 大气下陷 | 气逆之喘 |
|--------|----------|----------|
| 呼吸困难在 | **呼气**难 | **吸气**难 |
| 肩息（喘时耸肩） | 必**不**肩息 | 剧者必然肩息 |
| 脉象 | 沉迟微弱（寸关尤甚） | 多数、浮滑、尺弱寸强 |

## 六、大气下陷证候群（原著完整列举）

> "有呼吸短气者，有心中怔忡者，有淋漓大汗者，有神昏健忘者，有声颤身动者，有寒热往来者，有胸中满闷者，有努力呼吸似喘者，有咽干作渴者，有常常呵欠者，有肢体痿废者，有食后易饥者，有二便不禁者，有癃闭身肿者，有张口呼气外出而气不上达、肛门突出者，在女子有下血不止、更有经水逆行者。"

以上见症以心肺证候为主，常兼见脾胃证候。若单见脾胃证候而无心肺证候者，是谓中气下陷，非大气下陷。

## 七、冲气上冲证候群（原著原文）

> "冲气上冲，胃府之气亦失其息息下行之常，或亦转而上逆，阻塞饮食，不能下行，多化痰涎，因腹中膨闷、哕气、呃逆连连不止，甚则两肋疼胀、头目眩晕。其脉则弦硬而长。"

核心症状层级：
1. **特征性症状**：腹中有气自下上冲
2. **胃系症状**：胃脘满闷、哕气、呃逆、呕吐、饮食不下
3. **胸胁症状**：胸中烦热、胁下胀疼
4. **头面症状**：头目眩晕、气火挟痰上冲可致昏仆
5. **大便症状**：大便燥结（胃气不降，传送失职）
6. **脉象**：弦硬而长（右大于左为典型）

## 八、脏腑气化升降——辨证的解剖学基础

### 8.1 核心公理：肝升胃降

> "肝主左而宜升，胃主右而宜降。肝气不升，则先天之气化不能由肝上达；胃气不降，则后天之饮食不能由胃下输。此证之病根，正因当升者不升，当降者不降也。"

人体气机圆运动：
- **肝气左升** → 带动全身气化上行，为气化之始
- **胃气右降** → 带动饮食精微下行，息息下行为顺
- **胆气随胃降** → 少阳为游部，贯通经气
- **肺气肃降** → 与肝升相配合
- **肾气纳藏** → 为升降之根本

### 8.2 大气——统摄全局的核心

> "大气者，充满胸中，以司肺呼吸之气也。此气撑持全身，振作精神，心及心思、脑力、百骸动作，莫不赖于此。"

大气（宗气）：
- 以元气为根本，以水谷之气为养料，以胸中之地为宅窟
- 统摄三焦气化
- 大气一陷 → 全身气机失序 → 百症丛生
- 大气下陷不是单一症状，而是一个可表现为数十种不同症状的综合征

## 九、六经气化解读——对接仲景全库的唯一桥梁

张锡纯以气机升降视角重释六经，"六经实为十二经，以足赅手"。太阳经热力源自君相二火，胸中大气为枢纽。

**原著原文**（《六经总论》）：

> "推原其热力之由来，不外君、相二火。君火生于心之血脉与肺相循环，而散热于胸中大气以外通于营卫……相火生于肾中命门，肾原属水，中藏相火，其水火蒸热之气，由膀胱连三焦之脂膜以透达于身之外表……为太阳经之热力生于君、相二火，是以其经不但以膀胱为府，而亦以胸中为府。"

### 六经气化对应表

| 六经 | 气化解读 | 核心病机 | 治法方向 | 代表方 |
|------|----------|----------|----------|--------|
| **太阳** | 胸中大气+膀胱寒水 | 大气不足，卫外失固；或寒水郁闭 | 补助大气 | 桂枝汤 |
| **阳明** | 胃气不降+燥金之气 | 胃气当降不降，郁而化热化燥 | 清降阳明 | 白虎汤/承气汤 |
| **少阳** | 胆+三焦，一脉贯通（游部） | 枢机不利，气机出入不畅 | 调畅枢机 | 小柴胡汤 |
| **太阴** | 脾气不升 | 中焦气化不足，寒湿内停 | 温运中焦 | 理中汤 |
| **少阴** | 肾中水火 | 寒化：元阳不足；热化：阴虚阳浮 | 温阳或滋阴 | 四逆汤/黄连阿胶汤 |
| **厥阴** | 肝失疏泄 | 肝脏受邪，气机逆乱 | 调肝安冲 | 乌梅丸 |

### 太阳病桂枝汤的气化解

张锡纯认为桂枝汤治太阳中风的本质是**补助胸中大气**——太阳经热力依赖胸中大气外通营卫，大气不足则卫外失固，风邪乘虚而入。桂枝汤补益大气、调和营卫，非单纯解表。

## 十、辨证铁律（原著原则汇编）

1. **已病十之八九阴常不足**——滋阴保阴为先
2. **有力无力生死立判**——虚实明则治则无差
3. **弦硬多为假实**——弦直无起伏非真有力，误攻则危
4. **数脉主虚不主热**——数而无力大补元气，见数便用凉药"杀人无数"
5. **大气下陷呼吸气难在呼**——与气逆之喘（吸气难）为最关键的临床鉴别
6. **左右对比定升降**——右大于左降逆，左大于右平肝，左沉细升肝
7. **尺部有根方可治**——无根急固脱
8. **大气下陷非单一症状**——可表现为数十种症状的组合，需从整体气机把握
9. **升陷汤加减法**——气虚极加人参或寄生，加山萸肉防气涣散，升麻可增至4.5-6g
