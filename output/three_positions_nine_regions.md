## 交叉对比：三部同查

以下情况需要左右手同部对比 + 同手异部对比：

| 检查点 | 比较对象 | 意义 |
|--------|----------|------|
| 寸部对比 | 左寸 vs 右寸 | 左强→阴盛升法(吴茱萸汤)；右强→阳盛降法(越婢加半夏汤) |
| 关部对比 | 左关 vs 右关 | 左关强→气滞升法(小柴胡汤)；右关强→结胸降法(大黄黄连泻心汤) |
| 双关同强 | 左关弦+右关滑数 | **大柴胡汤**(双关弦) |
| 尺部对比 | 左尺 vs 右尺 | 左尺更塌→阴虚降法(百合地黄)；右尺更塌→阳虚升法(四逆) |
| 双尺同塌 | 左尺+右尺 | **肾气丸** |
| 寸尺反差 | 寸太过+尺不及 | **葛根汤**（左）或四逆内寒外热（右） |
| 浮位 vs 中位 | 同一部位深浅 | 浮位=表证(麻黄汤/桂枝汤)；中位=非表证(吴茱萸汤/越婢加半夏汤) ⚠️ 这是吴茱萸汤和麻黄汤的核心鉴别点！ |

## 使用方式

巍哥每次探查一步，我对应查表：

```
巍哥："左关中位弦，如弓弦按之不移"
  → 我查「左手关部·太过」表 → 候选：小柴胡汤 / 四逆散 / 苓桂术甘汤 / 大柴胡汤 / 乌梅丸 / 当归四逆汤

巍哥："总体摸着有力量，偏左手侧"
  → 排除四逆散/乌梅丸/当归四逆汤(总体不及)
  → 剩余：小柴胡汤 / 苓桂术甘汤 / 大柴胡汤

巍哥："右关正常，就左关独异"
  → 排除苓桂术甘汤(双关) / 大柴胡汤(双关)
  → 锁定：小柴胡汤
```

每一步都落到具体方，不绕路。
