## 七、知医邦28脉 ∈ 公式说明

置信度公式：`∈ = (阳性个数 × f × e - 阴性个数) / (阳性个数 + 阴性个数 + 1.5)`

- `{}` 包裹 = f=2（倍加，核心要素）
- `()` 包裹 = f=1/2（倍减，次要要素）
- 无括号 = f=1（普通阳性）
- e=1（当前版本固定）
- 阈值：∈>0 则脉象成立

## 八、舌诊辨证算法

**成立规则：**
- 同一选项满足3个分区 → 整体成立
- 同一分区满足4个选项 → 分区成立

**基础证型推导（∪∩组合）：**

| 证型 | 公式 | 证型 | 公式 |
|------|------|------|------|
| 表证 | AB∩BA | 实热 | BB |
| 寒证 | AE∪BA | 津亏 | AC∩BI |
| 热证 | AB∪AC∪AG∪BB | 水泛 | AE∩BH |
| 虚证 | AC∪AE∪AH∪BF∪CM | 痰滞 | BA∩BE |
| 实证 | AI∪BB∪BE | 燥结 | BB∩BE |
| 燥证 | AF∪BI∪CE | 虚寒 | AE |
| 湿证 | BH∪CD∪CM | 虚热 | AC |
| 亏证 | AA∪AF∪AH∪DE | 实寒 | AA∩BA |
| 滞证 | AD∪AI∪CL∪DA∪DB∪DC∪DD | | |

基于基础证型可进一步推导36证素（寒证、热证、虚寒、实寒、虚热、实热、阳盛、阳虚、阴盛、阴虚、寒湿、湿热、凉燥、燥热、虚证、实证、气虚、血虚、气滞、血瘀、亏证、滞证、燥证、湿证、水泛、湿滞、燥结、津亏、厥证、郁证、痰证、枯证、闭证、积证、劳证、脱证）。
