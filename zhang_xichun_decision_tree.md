# 张锡纯辨证方法论——原著提炼的完整决策树

> 源于《医学衷中参西录》原著，非网络解读。张锡纯辨证的根本方法是：以"阴常不足"为病机观，以气机升降为框架，以三部总看脉法定虚实，以左右对比定脏腑，以虚里尺部合参决预后，最终在自创方和仲景方两大方池中精准锁定。

---

## 一、根本病机观：阴常不足，阳常有余

**原著出处**：《驳方书贵阳抑阴论》

> "若论已病，又恒阳常有余，阴常不足……医者当调其阴阳，使之归于和平，或滋阴以化阳，或泻阳以保阴，其宜如此治者，又恒居十之八九。"

阳常有余非真阳盛——乃**阴虚不能敛阳，阳浮于外**。

治则铁律：
- 滋阴以化阳，泻阳以保阴
- 即使阳虚用桂附，亦必少佐滋阴——"上焦阴分不虚而后可受参芪，下焦阴分不虚而后可受桂附"
- 外感病中伤寒中于阴经宜热药者"百中无二三"
- 内伤病中虚劳、吐衄、淋痢、黄疸、肿胀、喘嗽，"阳盛阴虚者实为十之八九"

## 三、脉诊四铁律

| 铁律 | 内容 | 原著依据 |
|------|------|----------|
| **有力无力定虚实** | 三部总按，有力→实证/热证，无力→虚证/气陷 | "有力无力，虚实立判；虚实一明，治则无差" |
| **辨真假有力** | 真有力有洪滑之象（波涛叠涌、累累如贯珠），弦直无起伏为假实（阴虚阳浮），误攻则"凶危立见" | "若脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力" |
| **数脉主虚不主热** | 数而无力→大补元气，忌寒凉。数而有力方为真热。数至七八至不绝望——问气分能否支撑 | "见数脉便用凉药，杀人无数" |
| **革脉禁攻急敛** | 大而不洪 + 有力不滑 = 革脉，阴阳离绝之险。虽见便结腹满，**禁峻下**，先收敛固脱（萸肉龙骨），攻下则真气崩解 | "其形状如按鼓革，外虽硬而中空……虽大而不洪，虽有力而不滑" |

## 四、脉象-病机-方证完整对应

### 4.1 弦脉——核心脉象，主肝肾阴虚

弦是张锡纯最重视的脉象，集中体现"阴常不足"的根本病机。

| 弦脉细分 | 病机 | 临床表现 | 治法方向 |
|-----------|------|----------|----------|
| 左脉弦细无力 | 肝血虚、大气下陷 | 气短、乏力、产后下血、高年气虚血虚 | 升陷方向 |
| 左脉弦硬有力长 | 肝肾阴亏、阴虚不能潜阳、冲气上冲 | 吐血、咳逆、中风。兼数者阴血亏损甚 | 镇逆+滋阴 |
| 左脉弦细硬 + 右脉濡兼沉 | 湿痰留饮，中焦气化不足 | — | 化饮健脾 |
| 右脉弦细无力 | 土为木伤、脾胃失运 | 胁痛、胃脘痛 | 健脾+疏肝 |
| 右脉弦硬有力长（>左脉） | 冲气上冲、胃气不降 | 呃逆、呕吐、膈食、牙痛。重按不实者气血亏损 | 镇冲降胃 |
| 左右脉弦细无力 | 气血两亏、阴阳两虚 | 痢疾、肠炎。兼迟为寒饮结胸 | 双补气血 |
| 左右脉弦硬有力长 | 阴分有亏、肝木过盛 | 久痢阴虚、膈食 | 滋阴平肝 |
| 左脉平和微无力 + 右脉弦似有力 | 肾阴虚致冲气挟痰上冲 | — | 滋肾降冲 |
| 脉象和平，重按微弦无力 | **弦无力**，久病气化已衰 | 胁疼、不大便、外感迁延不愈 | 补升（忌破气） |

**原著原文**：

> "弦脉主肝血少、肝阴虚、肝火浮动。左脉弦，已阴虚不能潜阳，肝火浮动；右脉弦，已肝火与肝气并冲、胃上逆。兼数者热甚，兼沉者肝气郁。细而无力者气虚，硬而有力者虚火盛。"

**弦硬 vs 弦无力——关键鉴别**：

| | 弦硬 | 弦无力 |
|---|---|---|
| 本质 | 假有力，肝木横恣 | 真无力，气化已衰 |
| 脉象 | 弦直无起伏，硬而似有力 | 弦象犹在（肝气郁），但按之无力 |
| 治则 | **镇降**（赭石、龙骨、牡蛎） | **补升**（升陷、补气） |
| 错治后果 | 误攻则真气外泄 | 误用理气破气则气陷加重 |

## 五、大气下陷 vs 气逆之喘——原著关键鉴别

这是张锡纯独创的最重要临床鉴别，选错方向则"凶危立见"。

**原著原文**：

> "凡喘证，无论内伤外感，其剧者必然肩息；大气下陷者，虽至呼吸有声，必不肩息。盖肩息者，因喘者之吸气难；不肩息者，因大气下陷者之呼气难也。欲辨此证，可作呼气难与吸气难之状，以默自体验，临证自无差谬。又喘者之脉多数，或有浮滑之象，或尺弱寸强；大气下陷之脉，皆与此成反比例。"

| 鉴别点 | 大气下陷 | 气逆之喘 |
|--------|----------|----------|
| 呼吸困难在 | **呼气**难 | **吸气**难 |
| 肩息（喘时耸肩） | 必**不**肩息 | 剧者必然肩息 |
| 脉象 | 沉迟微弱（寸关尤甚） | 多数、浮滑、尺弱寸强 |

## 六、大气下陷证候群（原著完整列举）

> "有呼吸短气者，有心中怔忡者，有淋漓大汗者，有神昏健忘者，有声颤身动者，有寒热往来者，有胸中满闷者，有努力呼吸似喘者，有咽干作渴者，有常常呵欠者，有肢体痿废者，有食后易饥者，有二便不禁者，有癃闭身肿者，有张口呼气外出而气不上达、肛门突出者，在女子有下血不止、更有经水逆行者。"

以上见症以心肺证候为主，常兼见脾胃证候。若单见脾胃证候而无心肺证候者，是谓中气下陷，非大气下陷。

## 七、冲气上冲证候群（原著原文）

> "冲气上冲，胃府之气亦失其息息下行之常，或亦转而上逆，阻塞饮食，不能下行，多化痰涎，因腹中膨闷、哕气、呃逆连连不止，甚则两肋疼胀、头目眩晕。其脉则弦硬而长。"

核心症状层级：
1. **特征性症状**：腹中有气自下上冲
2. **胃系症状**：胃脘满闷、哕气、呃逆、呕吐、饮食不下
3. **胸胁症状**：胸中烦热、胁下胀疼
4. **头面症状**：头目眩晕、气火挟痰上冲可致昏仆
5. **大便症状**：大便燥结（胃气不降，传送失职）
6. **脉象**：弦硬而长（右大于左为典型）

## 八、脏腑气化升降——辨证的解剖学基础

### 8.1 核心公理：肝升胃降

> "肝主左而宜升，胃主右而宜降。肝气不升，则先天之气化不能由肝上达；胃气不降，则后天之饮食不能由胃下输。此证之病根，正因当升者不升，当降者不降也。"

人体气机圆运动：
- **肝气左升** → 带动全身气化上行，为气化之始
- **胃气右降** → 带动饮食精微下行，息息下行为顺
- **胆气随胃降** → 少阳为游部，贯通经气
- **肺气肃降** → 与肝升相配合
- **肾气纳藏** → 为升降之根本

### 8.2 大气——统摄全局的核心

> "大气者，充满胸中，以司肺呼吸之气也。此气撑持全身，振作精神，心及心思、脑力、百骸动作，莫不赖于此。"

大气（宗气）：
- 以元气为根本，以水谷之气为养料，以胸中之地为宅窟
- 统摄三焦气化
- 大气一陷 → 全身气机失序 → 百症丛生
- 大气下陷不是单一症状，而是一个可表现为数十种不同症状的综合征

### 8.3 虚里——大气搏动的体表窗口

> "虚里之络，即胃输水谷之气于胸中，以养大气之道路。而其贯膈络肺之余，又出于左乳下为动脉。是此动脉，当为大气之余波。"

- **健康人**：虚里搏动微弱不可见——大气盛而冲气不逆
- **冲气上冲**：虚里大动——"循虚里之大络……复出于左乳下与动脉相并，以致动脉因之大动"
- **革脉危证**：虚里大动 + 周身六脉弦硬——"周身脉管皆大动，几有破裂之势"
