### 第三线：温阳利水（真武汤/肾气丸/四逆汤集群）

## 四、三线协同关系

```
  上焦 ← 升陷汤（提大气）
   │
  中焦 ← 乌梅丸（调寒热·枢转厥阴）
   │
  下焦 ← 真武汤/肾气丸（温阳利水·固下元）
```

**协同逻辑**：
1. 升陷汤提上焦大气 → 解决'气短胸闷乏力'的直接痛苦
2. 乌梅丸调中焦枢机 → 解决'上热下寒'的格局性矛盾（左关弦·右尺弦）
3. 真武汤/肾气丸固下焦 → 解决'泡沫尿·双尺塌陷·阳虚水停'的根基问题

**禁忌交叉审查**：
- 升陷汤忌破气药 → 乌梅丸中细辛/干姜/桂枝偏辛散，与升陷汤的'提'方向一致，不冲突
- 乌梅丸含黄连黄柏苦寒 → 与郑钦安'阳虚阴盛'方向有张力。但厥阴病本就寒热错杂，少量苦寒反为'反佐'
- 真武汤芍药 → 张锡纯谓芍药苦平非苦寒，且真武汤中芍药'利小便+缓急+反佐附子燥烈'，与升陷不矛盾
