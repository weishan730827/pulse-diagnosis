# 张锡纯脉诊辨治——从理论到实战的系统教学

> 本文旨在提供一套可自学的张锡纯脉诊体系。先讲理论根基，再讲方法论，最后以原著医案验证每个核心观点。读完应该能理解"为什么这么摸脉"和"为什么这么用药"。

---

## 引言：张锡纯脉诊为什么值得单独学

在中国医学史上，张锡纯是少数几个真正把脉诊从经验层面推到理论层面的医家。他不是说"弦脉主肝"就完事了——他解释了弦脉背后的气机机制：**脉无起伏即弦，无起伏的原因是气机郁滞或大气虚陷，而气机郁滞的根源十之八九是阴虚不能涵阳。**

这套因果链贯穿他全部医案。学他的脉法，就是学他的辨证逻辑。

---

## 第一课：根本病机观——"阴常不足，阳常有余"

这是张锡纯一切诊断和用药的出发点。不理解这一条，后面所有脉象判断都会跑偏。

### 1.1 原著原文

> "若论已病，又恒阳常有余，阴常不足……医者当调其阴阳，使之归于和平，或滋阴以化阳，或泻阳以保阴，其宜如此治者，又恒居十之八九。"（《驳方书贵阳抑阴论》）

### 1.2 怎么理解

**阳常有余不是真阳盛**。试想一锅水（阴），下面烧火（阳）。如果水快烧干了，火没变大，但锅里的温度会急剧升高——表面上是"热"，本质是缺水。

张锡纯眼中的大多数热证、上火、高血压、吐血、中风，都是这个模型：

```
阴虚 →
  不能涵阳 → 阳浮于外（看起来像热证）→ 肝火浮动 →
    肝气横逆 → 冲气上冲 → 胃气不降 → 血随气逆（吐衄/中风/脑充血）
```

所以他反复强调：**滋阴以化阳，泻阳以保阴**。翻译：在锅里加水，火自然就不显得那么大了；实在火太大，可以泻一点火，但目的是保住锅里的水。

### 1.3 连桂附都要佐滋阴

> "上焦阴分不虚而后可受参芪，下焦阴分不虚而后可受桂附。"

即使是阳虚要用热药，前提也是**阴分不太虚**。阴分太虚的人，桂附一进去就"干烧"，越烧越干。这个原则他践行了一辈子。

## 第二课：脉象的核心——"有力"的真假之辨

这是张锡纯脉诊的第一道关口。过不了这道关，后面的辨证全错。

### 2.1 原著原文

> "脉之有力，有真有假。凡脉之真有力者，当于敦浓和缓中见之，此脾胃之气壮旺，能包括诸脏也。"

> "若脉象弦直，既无起伏之势，又无贯珠之形，虽大而有力，实非真有力之象。"

### 2.2 两个硬指标

张锡纯把"有力"拆成了两个可以摸出来的客观指标：

| 指标 | 感觉 | 代表什么 |
|------|------|----------|
| **洪象** | 指下像波涛一层层涌上来，有起伏感 | 气机通畅，血脉有推动力 |
| **滑象** | 指下圆润流畅，像珠子一颗颗滚过 | 津液充足，血脉有润滑 |

**真有力 = 洪 + 滑，两个都有。**

**假有力 = 脉大而硬，但指下是一条直线，没有起伏也没有滑润。**

### 2.3 为什么这个鉴别至关重要——医案

**《镇摄汤》证**：

一患者胸膈满闷，吃不下东西。前医看脉大而弦，按之有力，断定是实证，用了破气攻下——越治越重。

张锡纯接手，摸到的是：脉大而弦，按之似有力，但没有洪象（无起伏）、没有滑象（指下不圆润）。结论：这根本不是实证。这是**脾胃真气外泄，冲脉逆气上干**。脾胃自身的真气兜不住了往外泄，冲到上面堵住了胸膈。

治法完全相反：不是攻，是收。用镇摄汤（山萸肉、生龙骨、生牡蛎等收敛药），把外泄的真气收回来。胸膈自然通了。

**教训**：把假有力当成真有力，攻一次，伤一次根本。反复攻下去，真气散尽。

## 第三课：九种核心脉象的临床辨识

张锡纯论脉近三十种，但反复细讲的只有九种——因为这九种涵盖了"阴常不足"框架下的所有气机变化。

### 3.1 弦脉——最重要的脉象

弦脉在张锡纯笔下出现的频率远超其他八脉之和。原因是：弦 = 无起伏，而无起伏的原因要么是阴虚阳浮（肝火），要么是气陷不鼓（大气下陷）。这两者恰好覆盖了他病机论的全部。

**弦脉的关键不是"弦"这个标签，而是"弦成什么样"。**

#### 弦脉九种分型

| # | 左右 | 具体脉象 | 深层含义 | 治法方向 |
|---|------|----------|----------|----------|
| 1 | 左 | 弦细、无力 | 肝血虚 + 气陷。见于产后下血、肺劳痰喘。高年之人多此脉——气虚故无起伏，血虚故细而不濡 | 升陷+养血 |
| 2 | 左 | 弦硬、有力、长 | **注意：脉似有力而非真有力。** 肝肾阴亏，阴虚不能潜阳，肝火与冲气一起往上顶。这是中风前兆、吐血前兆 | 镇逆+滋阴，禁攻 |
| 3 | 左弦细硬 + 右濡沉 | 湿痰留饮堵在中焦，气化运转不了 | 化饮运中 |
| 4 | 右 | 弦细、无力 | 肝木横过来欺负脾胃（土为木伤），导致胁痛、胃痛。本质是肝的问题表现在右边 | 健脾中佐疏肝 |
| 5 | 右 | 弦硬、有力、长，且大于左脉 | **冲气上冲的标志性脉象。** "方书论脉有直上直下冲脉昭昭之语，所谓直上直下者，即脉弦且长之形状也。"右比左大说明冲胃之气全部上逆 | 降冲（赭石为主） |
| 6 | 左右 | 弦细、无力 | 气血两亏，阴阳两虚。兼迟者还有寒饮结胸 | 双补气血 |
| 7 | 左右 | 弦硬、有力、长 | 阴分亏损 + 肝木横恣。冲气上冲过甚，压迫胃气不能降 | 滋阴平肝降冲 |
| 8 | 左平和微无力 + 右弦似有力 | 表面看左脉还行，实际肾阴已虚，冲气挟痰上冲——这是一种隐蔽的上逆 | 滋肾降冲 |
| 9 | 脉象和平，重按微弦无力 | **弦无力——弦硬的对立面。** 久病气化已衰，弦象还在（肝气未舒），但完全无力（气化不能鼓脉） | **补升，忌破气** |

#### 弦硬 vs 弦无力——生死攸关的鉴别

这是一个张锡纯在无数案例中反复验证的对比：

| | 弦硬 | 弦无力 |
|---|---|---|
| 表象 | 硬邦邦，似乎有力 | 软绵绵，明显无力 |
| 本质 | 肝木横恣，假有力 | 气化已衰，真无力 |
| 感觉 | 弦直如拉紧的绳子 | 弦象存在但不硬，重按即软 |
| 治法 | **镇降收敛**（赭石、龙骨、牡蛎） | **补气升提**（黄芪、升麻、柴胡） |
| 错治 | 用补气升提→火上浇油，气更逆 | 用理气破气→气陷加重，人更虚 |

**弦无力医案**：

一患者不大便十日，饮食不纳数日。脉近和平，但重按微弦无力。前医都主张攻下通便。张锡纯力排众议：脉弦无力，是气化已衰，肠道无力推动，不是实热堵塞。攻下只会把人攻垮。

他用葱白熨法（外用温通）+ 赭石降逆（内服轻降），先恢复气化运转，大便自然通了。全程没有用任何泻药。

### 3.2 浮脉——浮在哪个部位是关键

| 部位 | 脉象 | 辨证 |
|------|------|------|
| 左 | 浮弦有力，但重按不实 | 肝血虚损，肝火上升，心虚不寐。阴亏火盛 |
| 右 | 浮芤 | 气血亏极，阳气已经飘起来了。兼数是阳气外越而发热——虚劳危象 |
| 左右 | 浮而无力 | 失血过多，下焦已经收不住了。兼数者阴阳两虚 |
| **关前浮 + 关后弱** | 浮在寸部，弱在尺部 | **桂枝汤证。** 浮≠单纯表证，是大气不足卫外失固，风邪乘虚。浮只是表层假象，重按之下弱脉仍在。所以桂枝汤要加黄芪补大气 |

**浮脉的核心教训**：浮脉不一定等于解表。张锡纯看到的浮脉，多半是阴虚阳浮、气血虚极。真正的风邪表证（桂枝汤证）反而是浮中带弱——大气不足才是根。

### 3.3 数脉——张锡纯挑战千年定论

> "世俗医者，遇脉数之证，大抵责之阴虚血涸。不知元气虚极莫支者，其脉可至极数。"

张锡纯自己做过实验：

> "理气药多于补气药，则脉即加数；补气药多于理气药，则脉即渐缓。是知脉之数与不数，固视乎血分之盈亏，实尤兼视乎气分之强弱。"

**数脉两条路**：

| 数而有力 | 真热 | 可清 |
|----------|------|------|
| **数而无力** | **气虚至极，撑不住了** | **大补元气，忌一切寒凉** |

**关键论点**：叔和说数至七八至是不治之脉——张锡纯反对。叔和是站在"阴虚血涸"的旧框架上说的。张锡纯加入"气分虚弱"维度后，很多七八至是可以治的——只要气分还没彻底垮，大补元气就能把脉拉回来。

**数脉医案**：

一外感患者迁延四十日，身酸软，动则眩晕，脉数逾五至，浮弦无力。前医反复用清热药，越治越差。张锡纯判断：表虽未解，正气已衰。脉浮弦说明表邪还在，但浮弦无力——这是**气分已撑不住了**，再清表就是杀人。他用扶正兼轻透，正气恢复后表邪自去。

### 3.4 微脉——大气下陷的体征信号

| 部位 | 脉象 | 辨证 |
|------|------|------|
| 左 | 微细、模糊、按之即无 | 肝胆虚热，或肝虚胁痛 |
| 右 | 微弱 | 气陷——纯粹的右边无力 |
| 左右两寸 | 微弱、细 | **大气下陷的标志性定位。** 兼迟者还有寒饮，兼沉者心肾不交 |

**关键点**：微脉一旦出现在两寸（关前），不管患者主诉什么症状，先排除大气下陷。大气下陷证候群极其庞杂——可以表现为气短、大汗、健忘、身颤、寒热往来、胸满、肢体痿废、二便不禁、甚至肛门突出、女子下血不止。任何一个症状单独看都不像"气陷"，但两寸微弱就是线索。

### 3.5 沉脉——关键是辨别真阳虚还是气陷

| 脉象 | 辨证 |
|------|------|
| 左沉弦 | 肝气郁滞不升 |
| **右沉迟微弱** | **大气下陷——兼迟者气陷又阳虚** |
| 左右沉细 | 心肺阳虚 |
| 伤寒脉沉细 | 多属阴证 |
| **温病脉沉细** | **多属阳证——这是关键鉴别** |
| 左右沉有力 | 实热郁于下焦，癃闭——真热藏在下面 |

---

### 3.6 迟脉——张锡纯对"迟=寒"的颠覆

> "脉之迟数，恒关于人身之热力，热力过盛则脉数，热力微弱则脉迟，此定理也。今此证虽有伏气化热，因大气下陷而脉仍迟。"

> "大气下陷之脉大抵皆迟，非因寒凉而迟也。"

**核心**：大气是推动血脉运行的动力源。动力不足，即使体内有热（伏热），脉照样快不起来。所以摸到迟脉，不能条件反射用热药——先摸寸关，看是不是大气下陷。

---

### 3.7 洪脉与滑脉

| 脉象 | 部位 | 辨证 |
|------|------|------|
| 洪长有力 | 右 | 胃气因热不降，血随气逆——吐衄 |
| 洪滑重按甚实 | 左右 | 阳明府实 + 心肝内热上攻 |
| 滑实 | 右 | 外感稽留肺胃 |
| 滑而有力 | 左右 | 湿热壅滞；或心气血虚损 + 心下停痰饮（惊悸不眠） |
| 滑而无力 | 左右 | 上盛下虚 |

## 第四课：大气下陷——张锡纯最独特的辨证体系

### 4.1 什么是大气

> "大气者，充满胸中，以司肺呼吸之气也……肺叶之辟阖，固为大气所司，而心机之跳动，亦为大气所司也。"

大气就是宗气，它管三件事：呼吸、心跳、脉搏起伏。大气一陷，这三样全出问题。

### 4.2 核心脉象：关前尤甚

> "其脉象沉迟微弱，关前尤甚。其剧者，或六脉不全，或参伍不调。"

**为什么必须摸关前（寸部）？** 因为胸中大气的位置就在寸关之上。大气下陷的第一定位信号永远是：**寸脉比关尺弱。**

### 4.3 兼证可以掩盖本脉

> "其脉多迟，而又间有数者，或因阴分虚损，或兼外感之热，或为热药所伤，乃兼证之现脉，非大气下陷之本脉也。"

这条非常重要——大气下陷的患者脉有时反而是数的，那是因为兼了阴虚或外热。不能因为脉数就排除大气下陷。

### 4.4 与阴虚气喘的鉴别——不能搞错方向

这是张锡纯强调了一辈子的鉴别：

| | 大气下陷之喘 | 阴虚不纳气之喘 |
|---|---|---|
| 呼吸困难 | **呼气难** | **吸气难** |
| 肩息（耸肩） | **不**肩息 | 剧者必然肩息 |
| 脉象 | 沉迟微弱，寸弱尺强（或尺略胜于寸） | 多数、浮滑，尺弱寸强 |
| 治法 | 升陷汤 | 滋阴纳气 |
| **错治后果** | 用降气治喘→气陷更深 | 用升陷→热更上冲 |

### 4.5 大气下陷医案

一患者胁疼、胸满、咳嗽，脉象和平但重按微弦无力。前医判断：胸胁满痛+咳 = 气郁，用理气药。结果越治越重，胸满加剧，发展到不能平卧。

张锡纯追问病史：患者感冒刚好就去下地干重活，出了一身大汗。感冒伤气在前，力田伤气在后，是大汗后气随汗泄——**这是大气下陷，不是气郁**。理气药把仅存的大气又破散了一部分。

改用升陷汤，数剂而愈。

**教训**：胸满可以是大气下陷的症状（大气陷下去堵在胸中，感觉像"满"），不能被"满"这个主观感觉误导。**脉重按微弦无力**才是真实信号。

## 第五课：三部二候——根与神的判断

前三课讲的是摸到什么脉，这一课判断：这个人的根本还撑不撑得住。

### 5.1 尺部候根——肾阴是生命的底牌

> "肝肾之阴为根。"

尺部代表肾阴的存亡。不管上面脉象多复杂，尺部没根，一切治疗都要围绕"固脱"转。

| 尺脉状态 | 判断 |
|----------|------|
| 重按有力 | 有根，能扛得住治疗 |
| 重按即无（尺脉无根） | 阴阳不维，危在顷刻 |
| 尺脉无根 + 寸部摇摇 | 即将脱 |
| 两尺按之即无 + 两寸浮甚 | 肾阴将竭，肝风已动——这是中风脱证的前兆 |
| 尺脉重按虚（左弦右濡） | 肾真阴虚损，真阳已经往上跑了 |
| 尺脉甚弱 | 阳升而阴不能应——上面在上冲，下面兜不住 |

### 5.2 脾胃候神——和缓就是生命力

> "脉之神，一在有力、无力，一在和缓。五行之中，土包括四行，六脉中皆和缓为贵，以其饶有脾土之气。即有神也。"

**和缓**是张锡纯最看重的脉象品质。任何脉象，只要还带着一丝和缓，说明脾胃之气还在濡养全身，这个人就还有转圜余地。一旦全无和缓——弦硬到底，毫无起伏——胃气已绝，治起来非常困难。

### 5.3 虚里——大气搏动的体表窗口

> "虚里之络，即胃输水谷之气于胸中，以养大气之道路。而其贯膈络肺之余，又出于左乳下为动脉。"

**虚里就是左乳头下方的心尖搏动区。** 张锡纯用它来印证脉诊：

- **健康人**：虚里搏动微弱不可见——大气充沛，冲气不逆
- **冲气上冲**：虚里大动——冲气沿着虚里的大络往上顶，和心尖搏动汇合，出现异常强烈的跳动
- **革脉危证**：虚里大动 + 周身六脉弦硬——全身脉管都在剧烈搏动，这是即将崩溃的信号

虚里是张锡纯的"双保险"——当脉象判断不确定时，摸一下左乳下就清楚了。

## 第六课：革脉与促脉——两个容易误判的特殊脉象

### 6.1 革脉：弦硬假有力的终极形态

革脉是张锡纯在医论中专篇论述的唯一脉象（原著第57篇）。定义：

> "其形状如按鼓革，外虽硬而中空，即弦脉之大而有力者。因其脉与弦脉相近，是以其脉虽大而不洪（无起伏故不洪），虽有力而不滑（中空故不滑）。"

革脉 = **大而不洪 + 有力而不滑** = 胃气已竭，阴阳离绝。

**革脉医案（张锡纯自记，三次转折）**：

初诊：患者脉大而硬，微兼洪象。张锡纯判断是外感余热未清，用了白虎加人参汤。

二诊：清完外热后，洪象消失了——但脉的大与硬丝毫未减，反而更明显。张锡纯意识到不对：洪象消了还有大硬，说明大硬不是热——是假象。

大便通下后病情陡变：脉的大与硬比之前加了两倍，周身脉管皆大动，几有破裂之势。张锡纯恍然大悟——这是革脉！外感热掩盖了本质，清下之后假有力崩解，真气将脱。

三诊：立即改为大剂收敛固脱——萸肉、龙骨、熟地、山药、野台参、白术、炙甘草。三剂后脉象渐平，续调而愈。

**教训**：
1. 革脉外观像实证（大而有力），本质上是最严重的虚证（胃气已竭）
2. **虽见便结腹满也绝对不能用攻下法**——攻下会让假有力崩解，真气散尽
3. 收敛固脱永远优先于通下攻邪

### 6.2 促脉：改步武而不止

张锡纯对促脉做了独特澄清，反驳"脉动速时一止"的传统说法：

> "譬如人之行路，行行且止，少停一步复行，是结、代也。又譬如人之奔驰，急急速走，路中偶遇不平，足下恒因有所龃龉，改其步武，而仍然奔驰不止，此促脉也。"

- **结脉/代脉**：走走停停——真停顿
- **促脉**：急速奔跑，步调偶尔乱了一下，但从来没停过——节奏紊乱但不中断

病因：**外热内陷，脉管拥挤**（热邪把脉管塞满了，所以跳得快且不规律）。

治法：**清热透表**（芩连类）。热透出来，脉管不拥挤了，促脉自消。这不是死证，不需要惊慌。

### 完整决策树

```
第一步：问主症 → 定气机方向

  左沉细 → 肝气不升，升肝

第四步：尺部候根 → 决预后
  有根 → 能治
  无根 → 急固脱

第五步：虚里印证 → 双保险
  虚里平静 → 大气充沛
  虚里大动+左脉弦长 → 冲气上冲
  虚里大动+六脉弦硬 → 革脉前兆
```

### 综合医案演示

一个完整案例走一遍五步：

患者：五十岁，头晕、时有呕恶、大便燥结三日未行。面色潮红，自觉"气从肚脐往上顶"。

**第一步（问主症）**：气从肚脐往上顶 → 上逆路径 → 冲气上冲。

**第二步（脉诊虚实）**：三部总按，脉来有力。进一步摸：有没有起伏？有。有没有滑润？有。→ 真有力，实热。不是假有力。

**第三步（左右对比）**：右脉弦硬而长，明显大于左脉；左脉也弦，但不如右脉硬。→ 右大于左 = 冲气上冲；左脉弦 = 兼肝火。治法：镇肝降胃，双管齐下。方选镇肝熄风汤。

**第四步（尺部）**：尺脉重按有力 → 有根，能治。

**第五步（虚里）**：左乳下搏动明显增强 → 印证了"冲气上冲"的判断。

治疗：镇肝熄风汤加减（牛膝引血下行、赭石降冲、龙骨牡蛎镇潜、白芍柔肝）。三剂后头晕减轻，大便自通。

**回头看**：这个病例为什么不用承气汤通便？因为大便燥结的原因不是阳明腑实——是冲气上冲导致胃气不降、肠道传送失职。降了冲气，胃气自然下行，大便就通了。这是张锡纯与常规辨证最大的不同。

## 第八课：总结——张锡纯脉诊的十三条铁律

1. **已病十之八九阴常不足**——滋阴保阴是总纲
2. **有力≠实证**——先看有没有洪滑，弦直无洪滑就是假有力
3. **弦硬 = 假有力 = 阴虚阳浮**——镇降收敛，不许攻
4. **弦无力 = 真无力 = 气化已衰**——补气升提，不许破气
5. **数脉主虚不主热**——数而无力用凉药就是杀人
6. **数至七八至不绝望**——先问气分能不能撑住
7. **迟脉未必寒**——先排除大气下陷
8. **关前微弱 = 大气下陷第一证据**——不管什么主诉
9. **大气下陷呼吸难在呼气**——与阴虚喘（吸气难）是生死鉴别
10. **右大于左 = 冲气上冲**——降冲为先
11. **尺部无根 = 要脱**——山萸肉，不许犹豫
12. **革脉禁攻急收敛**——大而不洪+有力不滑，虽便结腹满也不许攻
13. **促脉不是死证**——改步武而不止，清热透表可解
